# 登录认证接口

返回 [[00-接口总览]]。

## 本模块接口

- [[01-登录认证/01-手机号验证码登录|手机号验证码登录]]：手机号验证码登录，登录成功后返回用户信息并建立 App 登录态。
- [[01-登录认证/02-查询用户信息|查询用户信息]]：查询当前登录人的用户资料，用于启动恢复会话、我的页面和账号信息展示。
- [[01-登录认证/03-校验登录Token|校验登录 Token]]：校验本地 ECO_TOKEN 是否仍然有效，用于静默判断登录态。
- [[01-登录认证/04-退出登录|退出登录]]：退出登录并通知后端清理当前系统来源下的会话。
- [[01-登录认证/05-发送短信验证码|发送短信验证码]]：发送短信验证码，登录、绑定客户编码、注销账号等场景复用。
- [[01-登录认证/06-生成临时Token|生成临时 Token]]：生成临时 token，通常用于跳转 H5 或临时鉴权链路。
