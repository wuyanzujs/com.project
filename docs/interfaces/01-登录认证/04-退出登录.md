# 退出登录

返回 [[01-登录认证/00-模块总览|登录认证接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                   |
| --------- | ------------------------------------------------------ |
| 用途      | 退出登录并通知后端清理当前系统来源下的会话。           |
| 源码方法  | `authApi.logout`                                       |
| 源码位置  | `apps/com.deppon.app/src/services/auth/auth.api.ts:33` |
| HTTP 方法 | `POST`                                                 |
| 接口路径  | `/gwapi/userService/eco/user/secure/logout`            |
| 登录态    | 默认是                                                 |
| 请求类型  | `{ sysCode: string }`                                  |
| 响应类型  | `AppUser`                                              |

## 请求字段

类型：`{ sysCode: string }`。

| 字段      | 必填 | 类型     | 说明                                                |
| --------- | ---- | -------- | --------------------------------------------------- |
| `sysCode` | 是   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

类型：`AppUser`。

| 字段                | 必填 | 类型       | 说明                    |
| ------------------- | ---- | ---------- | ----------------------- |
| `id`                | 否   | `string`   | 数据主键或业务记录 ID。 |
| `mobile`            | 否   | `string`   | 手机号。                |
| `mobileEncrypt`     | 否   | `string`   | 加密后的手机号。        |
| `nickName`          | 否   | `string`   | 昵称。                  |
| `openId`            | 否   | `string`   | 微信或渠道 openId。     |
| `originalOpenId`    | 否   | `string`   | 原始 openId。           |
| `unionId`           | 否   | `string`   | 微信 unionId。          |
| `userName`          | 否   | `string`   | 用户名称。              |
| `avatarUrl`         | 否   | `string`   | 头像地址。              |
| `customerCode`      | 否   | `string`   | 客户编码。              |
| `contactCode`       | 否   | `string`   | 联系人编码。            |
| `allUserNames`      | 否   | `string[]` | 关联用户名列表。        |
| `thirdBindUserName` | 否   | `string`   | 第三方绑定用户名。      |

说明：该类型带有索引签名，后端可能返回额外字段，当前文档只展开源码中已声明的字段。
