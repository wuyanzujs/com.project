# 发送短信验证码

返回 [[01-登录认证/00-模块总览|登录认证接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 发送短信验证码，登录、绑定客户编码、注销账号等场景复用。 |
| 源码方法  | `authApi.sendSmsMessage`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/auth/auth.api.ts:40`   |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/messageService/eco/message/sendSmsMessage`       |
| 登录态    | 默认是                                                   |
| 请求类型  | `SendSmsRequest`                                         |
| 响应类型  | `无`                                                     |

## 请求字段

类型：`SendSmsRequest`。

| 字段          | 必填 | 类型                                       | 说明                                                |
| ------------- | ---- | ------------------------------------------ | --------------------------------------------------- |
| `mobile`      | 是   | `string`                                   | 手机号。                                            |
| `sysCode`     | 是   | `string`                                   | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `messageType` | 是   | 枚举："login"、"bindcuscode"、"canceluser" | 短信业务类型。                                      |

## 响应字段

无结构化字段。
