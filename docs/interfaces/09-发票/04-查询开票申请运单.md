# 查询开票申请运单

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 查询某个开票申请包含的运单和金额。                                    |
| 源码方法  | `invoiceService.queryHistoryWaybills`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:202`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `queryContainWaybill`                                                 |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `InvoiceHistoryWaybillRequest`                                        |
| 响应类型  | `InvoiceHistoryWaybillRaw`                                            |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

类型：`InvoiceHistoryWaybillRequest`。

| 字段      | 必填 | 类型     | 说明         |
| --------- | ---- | -------- | ------------ |
| `applyNo` | 是   | `string` | 发票申请号。 |

## 响应字段

响应类型：`{ [x: string]: string | number | null; }`。
