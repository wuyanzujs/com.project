# 提交电子发票申请

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 提交电子发票开票申请。                                                |
| 源码方法  | `invoiceService.submitApplyDraft`                                     |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:107`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `addTaskInfoByEle`                                                    |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `InvoiceApplySubmitRequest`                                           |
| 响应类型  | `InvoiceApplySubmitResult                                             | null` |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

类型：`InvoiceApplySubmitRequest`。

| 字段             | 必填 | 类型                       | 说明               |
| ---------------- | ---- | -------------------------- | ------------------ |
| `TaskDetailList` | 是   | `InvoiceApplyTaskDetail[]` | 开票任务明细列表。 |
| `TaskInfo`       | 是   | `InvoiceApplyTaskInfo`     | 开票任务主信息。   |

| 字段                        | 必填 | 类型      | 说明                                 |
| --------------------------- | ---- | --------- | ------------------------------------ |
| `payNo`                     | 是   | `string`  | 支付单号。                           |
| `payFlag`                   | 是   | `boolean` | 布尔或枚举标识字段。                 |
| `orderNo`                   | 是   | `string`  | 订单号。                             |
| `sourceBillNo`              | 是   | `string`  | 来源单号，当前开票场景通常是运单号。 |
| `amount`                    | 是   | `number`  | 金额或费用相关字段。                 |
| `unverAmount`               | 是   | `number`  | 未验证金额。                         |
| `paymentType`               | 是   | `string`  | 付款方式。                           |
| `electricSpecialTicketAuth` | 是   | `boolean` | 是否支持电子专票认证。               |
