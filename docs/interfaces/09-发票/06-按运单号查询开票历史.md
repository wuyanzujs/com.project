# 按运单号查询开票历史

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 按运单号查询开票历史。                                                |
| 源码方法  | `invoiceService.queryHistory`                                         |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:268`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `queryApplyByWayBillNo`                                               |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `InvoiceHistoryQueryRequest`                                          |
| 响应类型  | `InvoiceHistoryQueryResponse`                                         |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

类型：`InvoiceHistoryQueryRequest`。

| 字段           | 必填 | 类型     | 说明                                 |
| -------------- | ---- | -------- | ------------------------------------ |
| `sourceBillNo` | 是   | `string` | 来源单号，当前开票场景通常是运单号。 |

## 响应字段

类型：`InvoiceHistoryQueryResponse`，数组元素字段如下。

| 字段             | 必填 | 类型     | 说明                    |
| ---------------- | ---- | -------- | ----------------------- |
| `id`             | 是   | `number` | 数据主键或业务记录 ID。 |
| `acceptTinName`  | 否   | `string` | 购方抬头名称。          |
| `acceptTinCode`  | 否   | `string` | 购方税号。              |
| `applyNo`        | 否   | `string` | 发票申请号。            |
| `applyTime`      | 否   | `number` | 申请时间。              |
| `billAmount`     | 否   | `number` | 发票金额。              |
| `billCategory`   | 否   | `string` | 发票类型。              |
| `email`          | 否   | `string` | 邮箱地址。              |
| `elecLinkAdress` | 否   | `string` | 电子发票 PDF 地址。     |
| `remark`         | 否   | `string` | 备注。                  |
| `status`         | 否   | `string` | 状态。                  |
