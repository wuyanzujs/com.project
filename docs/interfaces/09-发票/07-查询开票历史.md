# 查询开票历史

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 分页查询开票历史。                                                    |
| 源码方法  | `invoiceService.queryHistory`                                         |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:292`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `queryInvoiceHistory`                                                 |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `InvoiceHistoryListRequest`                                           |
| 响应类型  | `InvoiceHistoryListResponse`                                          |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

类型：`InvoiceHistoryListRequest`。

| 字段                | 必填 | 类型     | 说明           |
| ------------------- | ---- | -------- | -------------- |
| `currentPage`       | 是   | `number` | 当前页码。     |
| `pageSize`          | 是   | `number` | 每页条数。     |
| `openBillStartDate` | 是   | `string` | 开票开始日期。 |
| `openBillEndDate`   | 是   | `string` | 开票结束日期。 |

## 响应字段

类型：`InvoiceHistoryListResponse`。

| 字段        | 必填 | 类型                  | 说明                                  |
| ----------- | ---- | --------------------- | ------------------------------------- |
| `pageIndex` | 是   | `number`              | 页码，从当前业务传参看通常从 1 开始。 |
| `pageSize`  | 是   | `number`              | 每页条数。                            |
| `total`     | 是   | `number`              | 总记录数。                            |
| `list`      | 是   | `InvoiceHistoryRaw[]` | 列表数据。                            |

### `InvoiceHistoryListResponse.list[]` 字段

| 字段             | 必填 | 类型     | 说明                    |
| ---------------- | ---- | -------- | ----------------------- |
| `id`             | 是   | `number` | 数据主键或业务记录 ID。 |
| `acceptTinName`  | 否   | `string` | 购方抬头名称。          |
| `acceptTinCode`  | 否   | `string` | 购方税号。              |
| `applyNo`        | 否   | `string` | 发票申请号。            |
| `applyTime`      | 否   | `number` | 申请时间。              |
| `billAmount`     | 否   | `number` | 发票金额。              |
| `billCategory`   | 否   | `string` | 发票类型。              |
| `email`          | 否   | `string` | 邮箱地址。              |
| `elecLinkAdress` | 否   | `string` | 电子发票 PDF 地址。     |
| `remark`         | 否   | `string` | 备注。                  |
| `status`         | 否   | `string` | 状态。                  |
