# 查询发票抬头

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 查询发票抬头列表。                                                    |
| 源码方法  | `invoiceService.queryTaxpayers`                                       |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:324`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `queryTaxpayerInfo`                                                   |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `无`                                                                  |
| 响应类型  | `InvoiceTaxpayerRaw[]`                                                |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

无请求体。

## 响应字段

类型：`InvoiceTaxpayerRaw[]`，数组元素字段如下。

| 字段            | 必填 | 类型                  | 说明                    |
| --------------- | ---- | --------------------- | ----------------------- |
| `id`            | 是   | `number`              | 数据主键或业务记录 ID。 |
| `acceptTinCode` | 否   | `string`              | 购方税号。              |
| `acceptTinName` | 否   | `string`              | 购方抬头名称。          |
| `address`       | 否   | `string`              | 详细地址。              |
| `bankNo`        | 否   | `string`              | 银行账号。              |
| `phone`         | 否   | `string`              | 联系电话。              |
| `customerType`  | 否   | `InvoiceTaxpayerType` | 客户类型。              |
| `isDefault`     | 否   | 枚举："0"、"1"        | 是否默认抬头。          |
| `openBank`      | 否   | `string`              | 开户行。                |
| `remark`        | 否   | `string`              | 备注。                  |
