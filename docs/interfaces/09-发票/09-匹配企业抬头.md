# 匹配企业抬头

返回 [[09-发票/00-模块总览|发票接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                  |
| --------- | --------------------------------------------------------------------- |
| 用途      | 根据企业名称匹配税号、开户地址、开户行等企业抬头信息。                |
| 源码方法  | `invoiceService.queryTaxpayerMatches`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/invoice/invoice.service.ts:354`     |
| HTTP 方法 | `POST`                                                                |
| 发票 path | `queryCustomerTaxName`                                                |
| 接口路径  | `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService` |
| 登录态    | 是                                                                    |
| 请求类型  | `InvoiceTaxpayerMatchRequest`                                         |
| 响应类型  | `InvoiceTaxpayerMatch[]`                                              |

## 调用说明

这个接口走发票公共网关，真实 HTTP path 固定为 `/gwapi/onlineService/eco/online/invoice/secure/invoiceCommonService`，业务动作由请求体里的 `path` 区分。字段表只展开 `data` 部分，公共外层结构是 `{ path, data }`。

## 请求字段

类型：`InvoiceTaxpayerMatchRequest`。

| 字段           | 必填 | 类型           | 说明           |
| -------------- | ---- | -------------- | -------------- |
| `taxName`      | 是   | `string`       | 企业抬头名称。 |
| `customerType` | 是   | 枚举："0"、"1" | 客户类型。     |

## 响应字段

类型：`InvoiceTaxpayerMatch[]`，数组元素字段如下。

| 字段            | 必填 | 类型     | 说明           |
| --------------- | ---- | -------- | -------------- |
| `taxName`       | 是   | `string` | 企业抬头名称。 |
| `taxNo`         | 是   | `string` | 企业税号。     |
| `taxAddress`    | 是   | `string` | 税务注册地址。 |
| `taxTelephone`  | 是   | `string` | 税务登记电话。 |
| `taxBankName`   | 是   | `string` | 开户行名称。   |
| `taxBankNumber` | 是   | `string` | 银行账号。     |
| `customerType`  | 是   | `number` | 客户类型。     |
