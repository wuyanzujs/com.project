# 查询乡镇列表

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 根据省市区查询乡镇列表。                                     |
| 源码方法  | `contactApi.queryTownList`                                   |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:98` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/queryService/eco/query/city/queryTownByCounty`       |
| 登录态    | 否                                                           |
| 请求类型  | `ContactTownListRequest`                                     |
| 响应类型  | `ContactTownItem[]`                                          |

## 请求字段

类型：`ContactTownListRequest`。

| 字段           | 必填 | 类型     | 说明       |
| -------------- | ---- | -------- | ---------- |
| `provinceName` | 是   | `string` | 省份名称。 |
| `cityName`     | 是   | `string` | 城市名称。 |
| `countyName`   | 是   | `string` | 区县名称。 |

## 响应字段

类型：`ContactTownItem[]`，数组元素字段如下。

| 字段           | 必填 | 类型     | 说明             |
| -------------- | ---- | -------- | ---------------- |
| `provinceName` | 是   | `string` | 省份名称。       |
| `provinceCode` | 是   | `string` | 省份编码。       |
| `cityName`     | 是   | `string` | 城市名称。       |
| `cityCode`     | 是   | `string` | 城市编码。       |
| `countyName`   | 是   | `string` | 区县名称。       |
| `countyCode`   | 是   | `string` | 区县编码。       |
| `townName`     | 是   | `string` | 乡镇或街道名称。 |
| `townCode`     | 是   | `string` | 乡镇或街道编码。 |
