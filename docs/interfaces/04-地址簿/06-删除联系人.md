# 删除联系人

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 按联系人 id 删除地址簿记录。                                 |
| 源码方法  | `contactApi.deleteContacts`                                  |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:41` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/userService/eco/user/contact/secure/deleteContact`   |
| 登录态    | 默认是                                                       |
| 请求类型  | `{ ids: string[] }`                                          |
| 响应类型  | `无`                                                         |

## 请求字段

类型：`{ ids: string[] }`。

| 字段  | 必填 | 类型       | 说明                     |
| ----- | ---- | ---------- | ------------------------ |
| `ids` | 是   | `string[]` | 待删除的联系人 ID 列表。 |

## 响应字段

无结构化字段。
