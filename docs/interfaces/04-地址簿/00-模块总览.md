# 地址簿接口

返回 [[00-接口总览]]。

## 本模块接口

- [[04-地址簿/01-查询联系人数量|查询联系人数量]]：查询地址簿联系人总数。
- [[04-地址簿/02-校验详细地址|校验详细地址]]：校验地址是否足够详细。
- [[04-地址簿/03-按定位推荐联系人|按定位推荐联系人]]：根据定位经纬度和地址推荐最近联系人地址信息。
- [[04-地址簿/04-查询地址簿|查询地址簿]]：分页查询地址簿联系人列表。
- [[04-地址簿/05-新增或更新联系人|新增或更新联系人]]：新增或更新地址簿联系人，是否更新由联系人 id 决定。
- [[04-地址簿/06-删除联系人|删除联系人]]：按联系人 id 删除地址簿记录。
- [[04-地址簿/07-批量删除联系人|批量删除联系人]]：按删除类型批量清理地址簿联系人。
- [[04-地址簿/08-查询默认联系人|查询默认联系人]]：查询默认联系人，用于寄件页快速填充寄件人或收件人。
- [[04-地址簿/09-智能解析地址|智能解析地址]]：解析自然语言地址，识别姓名、电话、地址等信息。
- [[04-地址簿/10-解析四级地址|解析四级地址]]：解析地址并返回四级地址编码。
- [[04-地址簿/11-查询地址联想|查询地址联想]]：根据省市区和详细地址查询地址联想建议。
- [[04-地址簿/12-查询乡镇列表|查询乡镇列表]]：根据省市区查询乡镇列表。
