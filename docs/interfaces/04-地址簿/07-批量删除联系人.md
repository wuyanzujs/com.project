# 批量删除联系人

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                            |
| --------- | --------------------------------------------------------------- |
| 用途      | 按删除类型批量清理地址簿联系人。                                |
| 源码方法  | `contactApi.batchDeleteContact`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:47`    |
| HTTP 方法 | `POST`                                                          |
| 接口路径  | `/gwapi/userService/eco/user/contact/secure/batchDeleteContact` |
| 登录态    | 默认是                                                          |
| 请求类型  | `{ deleteType: number }`                                        |
| 响应类型  | `number`                                                        |

## 请求字段

类型：`{ deleteType: number }`。

| 字段         | 必填 | 类型     | 说明           |
| ------------ | ---- | -------- | -------------- |
| `deleteType` | 是   | `number` | 批量删除类型。 |

## 响应字段

响应类型：`number`。
