# 智能解析地址

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                         |
| --------- | ---------------------------------------------------------------------------- |
| 用途      | 解析自然语言地址，识别姓名、电话、地址等信息。                               |
| 源码方法  | `contactApi.analyzeAddress`                                                  |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:65`                 |
| HTTP 方法 | `POST`                                                                       |
| 接口路径  | `/gwapi/onlineService/eco/online/addressAiResolutionApi/addressAiResolution` |
| 登录态    | 默认是                                                                       |
| 请求类型  | `{ address: string }`                                                        |
| 响应类型  | `ContactAnalysis`                                                            |

## 请求字段

类型：`{ address: string }`。

| 字段      | 必填 | 类型     | 说明       |
| --------- | ---- | -------- | ---------- |
| `address` | 是   | `string` | 详细地址。 |

## 响应字段

类型：`ContactAnalysis`。

| 字段          | 必填 | 类型           | 说明               |
| ------------- | ---- | -------------- | ------------------ |
| `proCityName` | 是   | string 或 null | 名称字段。         |
| `town`        | 是   | string 或 null | 乡镇或街道名称。   |
| `address`     | 是   | string 或 null | 详细地址。         |
| `name`        | 是   | string 或 null | 姓名或名称。       |
| `telephone`   | 是   | string 或 null | 手机号或联系电话。 |
| `addressType` | 是   | string 或 null | 地址字段。         |
| `extension`   | 是   | string 或 null | 电话分机号。       |
