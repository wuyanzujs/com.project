# 新增或更新联系人

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                                                                     |
| --------- | ------------------------------------------------------------------------------------------------------------------------ |
| 用途      | 新增或更新地址簿联系人，是否更新由联系人 id 决定。                                                                       |
| 源码方法  | `contactApi.saveContact`                                                                                                 |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:29`                                                             |
| HTTP 方法 | `POST`                                                                                                                   |
| 接口路径  | `/gwapi/userService/eco/user/contact/secure/insertContact`<br>`/gwapi/userService/eco/user/contact/secure/updateContact` |
| 登录态    | 默认是                                                                                                                   |
| 请求类型  | `Contact`                                                                                                                |
| 响应类型  | `Contact`                                                                                                                |

## 请求字段

类型：`Contact`。

| 字段             | 必填 | 类型                  | 说明                    |
| ---------------- | ---- | --------------------- | ----------------------- |
| `id`             | 否   | `string`              | 数据主键或业务记录 ID。 |
| `name`           | 是   | `string`              | 姓名或名称。            |
| `telephone`      | 是   | `string`              | 手机号或联系电话。      |
| `fixedPhone`     | 否   | `string`              | 固定电话。              |
| `extension`      | 否   | `string`              | 电话分机号。            |
| `province`       | 是   | `string`              | 省份名称。              |
| `city`           | 是   | `string`              | 城市名称。              |
| `county`         | 是   | `string`              | 区县名称。              |
| `town`           | 否   | `string`              | 乡镇或街道名称。        |
| `address`        | 是   | `string`              | 详细地址。              |
| `company`        | 否   | `string`              | 公司名称。              |
| `type`           | 是   | 枚举：0、1            | 业务类型。              |
| `defaultAddress` | 是   | 枚举："0"、"1"        | 是否默认地址。          |
| `regionType`     | 是   | 枚举：空字符串、"GAT" | 区域类型，例如港澳台。  |
| `saveStatus`     | 否   | `boolean`             | 保存状态。              |
| `saveToBook`     | 否   | `boolean`             | 是否保存到地址簿。      |
