# 查询地址簿

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 分页查询地址簿联系人列表。                                   |
| 源码方法  | `contactApi.queryContact`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:18` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/userService/eco/user/contact/secure/queryContact`    |
| 登录态    | 默认是                                                       |
| 请求类型  | `ContactListRequest`                                         |
| 响应类型  | `ContactListResponse`                                        |

## 请求字段

类型：`ContactListRequest`。

| 字段            | 必填 | 类型                  | 说明                                                |
| --------------- | ---- | --------------------- | --------------------------------------------------- |
| `pageIndex`     | 是   | `number`              | 页码，从当前业务传参看通常从 1 开始。               |
| `pageSize`      | 是   | `number`              | 每页条数。                                          |
| `filterContent` | 是   | `string`              | 搜索关键字。                                        |
| `regionType`    | 是   | 枚举：空字符串、"GAT" | 区域类型，例如港澳台。                              |
| `sysCode`       | 是   | `string`              | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

类型：`ContactListResponse`。

| 字段        | 必填 | 类型        | 说明       |
| ----------- | ---- | ----------- | ---------- |
| `list`      | 是   | `Contact[]` | 列表数据。 |
| `pageNum`   | 是   | `number`    | 当前页码。 |
| `pageSize`  | 是   | `number`    | 每页条数。 |
| `totalPage` | 是   | `number`    | 总页数。   |
| `totalRows` | 是   | `number`    | 总记录数。 |

### `ContactListResponse.list[]` 字段

| 字段             | 必填 | 类型                  | 说明                    |
| ---------------- | ---- | --------------------- | ----------------------- |
| `id`             | 否   | `string`              | 数据主键或业务记录 ID。 |
| `name`           | 是   | `string`              | 姓名或名称。            |
| `telephone`      | 是   | `string`              | 手机号或联系电话。      |
| `fixedPhone`     | 否   | `string`              | 固定电话。              |
| `extension`      | 否   | `string`              | 电话分机号。            |
| `province`       | 是   | `string`              | 省份名称。              |
| `city`           | 是   | `string`              | 城市名称。              |
| `county`         | 是   | `string`              | 区县名称。              |
| `town`           | 否   | `string`              | 乡镇或街道名称。        |
| `address`        | 是   | `string`              | 详细地址。              |
| `company`        | 否   | `string`              | 公司名称。              |
| `type`           | 是   | 枚举：0、1            | 业务类型。              |
| `defaultAddress` | 是   | 枚举："0"、"1"        | 是否默认地址。          |
| `regionType`     | 是   | 枚举：空字符串、"GAT" | 区域类型，例如港澳台。  |
| `saveStatus`     | 否   | `boolean`             | 保存状态。              |
| `saveToBook`     | 否   | `boolean`             | 是否保存到地址簿。      |
