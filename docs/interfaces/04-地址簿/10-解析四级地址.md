# 解析四级地址

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                             |
| --------- | -------------------------------------------------------------------------------- |
| 用途      | 解析地址并返回四级地址编码。                                                     |
| 源码方法  | `contactApi.analyzeAddress4`                                                     |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:76`                     |
| HTTP 方法 | `POST`                                                                           |
| 接口路径  | `/gwapi/onlineService/eco/online/addressAiResolutionApi/addressCodingDirections` |
| 登录态    | 默认是                                                                           |
| 请求类型  | `ContactAnalysis4Request`                                                        |
| 响应类型  | `ContactAnalysis4`                                                               |

## 请求字段

类型：`ContactAnalysis4Request`。

| 字段            | 必填 | 类型     | 说明       |
| --------------- | ---- | -------- | ---------- |
| `province`      | 否   | `string` | 省份名称。 |
| `city`          | 否   | `string` | 城市名称。 |
| `county`        | 否   | `string` | 区县名称。 |
| `detailAddress` | 是   | `string` | 详细地址。 |

## 响应字段

类型：`ContactAnalysis4`。

| 字段            | 必填 | 类型     | 说明             |
| --------------- | ---- | -------- | ---------------- |
| `province`      | 是   | `string` | 省份名称。       |
| `provinceCode`  | 是   | `string` | 省份编码。       |
| `city`          | 是   | `string` | 城市名称。       |
| `cityCode`      | 是   | `string` | 城市编码。       |
| `county`        | 是   | `string` | 区县名称。       |
| `countyCode`    | 是   | `string` | 区县编码。       |
| `town`          | 是   | `string` | 乡镇或街道名称。 |
| `townCode`      | 是   | `string` | 乡镇或街道编码。 |
| `detailAddress` | 是   | `string` | 详细地址。       |
| `addressType`   | 是   | `string` | 地址字段。       |
