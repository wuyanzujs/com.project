# 查询地址联想

返回 [[04-地址簿/00-模块总览|地址簿接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 根据省市区和详细地址查询地址联想建议。                       |
| 源码方法  | `contactApi.queryAddressHints`                               |
| 源码位置  | `apps/com.deppon.app/src/services/contact/contact.api.ts:87` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/queryService/eco/query/addressImagine`               |
| 登录态    | 默认是                                                       |
| 请求类型  | `ContactQueryRequest`                                        |
| 响应类型  | `ContactQueryResponse`                                       |

## 请求字段

类型：`ContactQueryRequest`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `province` | 是   | `string` | 省份名称。 |
| `city`     | 是   | `string` | 城市名称。 |
| `county`   | 是   | `string` | 区县名称。 |
| `address`  | 是   | `string` | 详细地址。 |

## 响应字段

响应类型：`string[]`。
