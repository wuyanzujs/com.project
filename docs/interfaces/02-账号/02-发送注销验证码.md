# 发送注销验证码

返回 [[02-账号/00-模块总览|账号接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                        |
| --------- | ----------------------------------------------------------- |
| 用途      | 发送注销账号短信验证码。                                    |
| 源码方法  | `accountApi.sendCancelSms`                                  |
| 源码位置  | `apps/com.deppon.app/src/services/account/account.api.ts:7` |
| HTTP 方法 | `POST`                                                      |
| 接口路径  | `/gwapi/messageService/eco/message/sendSmsMessage`          |
| 登录态    | 默认是                                                      |
| 请求类型  | `SendSmsRequest`                                            |
| 响应类型  | `无`                                                        |

## 请求字段

类型：`SendSmsRequest`。

| 字段          | 必填 | 类型                                       | 说明                                                |
| ------------- | ---- | ------------------------------------------ | --------------------------------------------------- |
| `mobile`      | 是   | `string`                                   | 手机号。                                            |
| `sysCode`     | 是   | `string`                                   | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `messageType` | 是   | 枚举："login"、"bindcuscode"、"canceluser" | 短信业务类型。                                      |

## 响应字段

无结构化字段。
