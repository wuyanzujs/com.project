# 注销账号

返回 [[02-账号/00-模块总览|账号接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 提交账号注销申请，后端校验验证码、手机号和注销校验方式。     |
| 源码方法  | `accountApi.cancelAccount`                                   |
| 源码位置  | `apps/com.deppon.app/src/services/account/account.api.ts:14` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/userService/eco/user/secure/cancel`                  |
| 登录态    | 默认是                                                       |
| 请求类型  | `AccountCancelRequest`                                       |
| 响应类型  | `boolean`                                                    |

## 请求字段

类型：`AccountCancelRequest`。

| 字段        | 必填 | 类型       | 说明                                                |
| ----------- | ---- | ---------- | --------------------------------------------------- |
| `code`      | 是   | `string`   | 编码。                                              |
| `mobile`    | 是   | `string`   | 手机号。                                            |
| `password`  | 是   | `string`   | 密码或后端校验用凭证。                              |
| `sysCode`   | 是   | `string`   | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `validType` | 是   | 枚举：1、2 | 校验方式类型。                                      |

## 响应字段

响应类型：`boolean`。
