## 接口说明

| 项目      | 说明                                                                                         |
| --------- | -------------------------------------------------------------------------------------------- |
| 用途      | App 内 WebView 承接入口：在线理赔。这不是 depponHttp JSON 接口，而是 H5 页面或文件预览入口。 |
| 源码方法  | `APP_WEB_TARGETS.SUPPORT_CLAIM`                                                              |
| 源码位置  | `apps/com.deppon.app/src/shared/webview/appWeb.ts:97`                                        |
| HTTP 方法 | `WEBVIEW`                                                                                    |
| 接口路径  | `/depponmobile/h5/index#/claimPackagePages/index`                                            |
| 登录态    | 默认是                                                                                       |
| 请求类型  | `URL 参数按调用方拼接`                                                                       |
| 响应类型  | `H5 页面`                                                                                    |
