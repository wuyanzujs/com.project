# 查询待支付费用

返回 [[08-支付/00-模块总览|支付接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                 |
| --------- | -------------------------------------------------------------------- |
| 用途      | 查询待核销或待支付费用列表。                                         |
| 源码方法  | `paymentApi.queryUnpaidList`                                         |
| 源码位置  | `apps/com.deppon.app/src/services/payment/payment.api.ts:9`          |
| HTTP 方法 | `POST`                                                               |
| 接口路径  | `/gwapi/onlineService/eco/online/pay/pmc/secure/queryUnWriteOffList` |
| 登录态    | 默认是                                                               |
| 请求类型  | `PaymentListRequest`                                                 |
| 响应类型  | `PaymentListResponse`                                                |

## 请求字段

类型：`PaymentListRequest`。

| 字段               | 必填 | 类型               | 说明                                  |
| ------------------ | ---- | ------------------ | ------------------------------------- |
| `orderSource`      | 是   | `string`           | 订单来源。                            |
| `startTime`        | 是   | `string`           | 开始时间。                            |
| `endTime`          | 是   | `string`           | 结束时间。                            |
| `pageIndex`        | 是   | `number`           | 页码，从当前业务传参看通常从 1 开始。 |
| `pageSize`         | 是   | `number`           | 每页条数。                            |
| `writeOffStatus`   | 是   | 枚举：0、1         | 核销状态。                            |
| `waybillNo`        | 否   | `string`           | 运单号。                              |
| `customerProperty` | 否   | `PaymentOrderType` | 客户或账单属性。                      |

## 响应字段

类型：`PaymentListResponse`。

| 字段        | 必填 | 类型                    | 说明                 |
| ----------- | ---- | ----------------------- | -------------------- |
| `errorMsg`  | 是   | string 或 null          | 错误信息。           |
| `isError`   | 是   | 枚举："YES"、"NO"、null | 布尔或枚举标识字段。 |
| `pageNum`   | 是   | number 或 null          | 当前页码。           |
| `pageSize`  | 是   | number 或 null          | 每页条数。           |
| `totalRows` | 是   | number 或 null          | 总记录数。           |
| `totalPage` | 是   | number 或 null          | 总页数。             |
| `list`      | 是   | PaymentItem[] 或 null   | 列表数据。           |
