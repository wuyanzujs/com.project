# 查询待打印订单列表

返回 [[13-打印/00-模块总览|打印接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目 | 说明 |
| --- | --- |
| 用途 | 分页查询待打印或已打印面单，并获取对应日期范围内的 tab 数量。 |
| 源码方法 | `printApi.queryList`、`printService.queryList`、`printService.queryCounts` |
| 源码位置 | `apps/com.deppon.app/src/services/print/print.api.ts`、`print.service.ts` |
| HTTP 方法 | `POST` |
| 接口路径 | `/gwapi/onlineService/eco/online/print/order/secure/queryNewOrderPrintList` |
| 登录态 | 是，使用统一 OWS `ECO_TOKEN` 会话。 |
| 请求类型 | `PrintListRequest` |
| 响应类型 | `PrintListResponse` |

## 请求字段

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `pageNum` | `number` | 是 | 页码，从 1 开始。 |
| `pageSize` | `number` | 是 | 每页数量；列表默认 10，tab 计数查询为 1。 |
| `startTime` | `string` | 是 | 开始时间，格式 `yyyy-MM-dd 00:00:00`。 |
| `endTime` | `string` | 是 | 结束时间，格式 `yyyy-MM-dd 23:59:59`。 |
| `searchType` | `'1' \| '2'` | 是 | `1` 待打印，`2` 已打印。 |

页面提供今天、近 3 天、近一周、近一个月和近三个月五档范围，默认近 3 天；范围按本地日历日包含今天计算。

## 响应字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `pageNum` | `number` | 当前页码。 |
| `pageSize` | `number` | 每页数量。 |
| `totalPage` | `number` | 总页数。 |
| `totalRows` | `number` | 总记录数，也用于 tab 数量。 |
| `list` | `PrintOrderRaw[]` | 当前页打印订单。 |

### `PrintOrderRaw`

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 打印记录 ID。 |
| `waybillNumber` | `string` | 运单号。 |
| `receiveName` | `string` | 收件人姓名。 |
| `receivePhone` | `string` | 收件人手机号；进入页面 VM 前会脱敏。 |
| `receiveProvince` | `string` | 收件省。 |
| `receiveCity` | `string` | 收件市。 |
| `receiveArea` | `string` | 收件区县。 |
| `receiveAddress` | `string` | 收件详细地址。 |
| `deviceType` | `'1' \| '2'` | 可选设备类型，当前只保留字段，不驱动打印。 |

## App 边界

- `pages/print/list` 只调用 `printService`，不直接访问 API。
- 待打印和已打印数量使用两个独立查询；单侧失败显示 `--`，不会清空另一侧数量或阻断列表。
- 当前不调用蓝牙、模板、打印配置和状态回写接口，也不提供勾选或打印按钮。
