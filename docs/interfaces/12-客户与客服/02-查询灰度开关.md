# 查询灰度开关

返回 [[12-客户与客服/00-模块总览|客户和客服接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                        |
| --------- | ----------------------------------------------------------- |
| 用途      | 读取灰度开关配置，用于客服、调研等功能开关。                |
| 源码方法  | `supportApi.fetchSwitchConfig`                              |
| 源码位置  | `apps/com.deppon.app/src/services/support/support.api.ts:4` |
| HTTP 方法 | `POST`                                                      |
| 接口路径  | `/gwapi/queryService/eco/query/switch/graySwitch`           |
| 登录态    | 否                                                          |
| 请求类型  | `{ switchKey: string }`                                     |
| 响应类型  | `TResult`                                                   |

## 请求字段

类型：`{ switchKey: string }`。

| 字段        | 必填 | 类型     | 说明         |
| ----------- | ---- | -------- | ------------ |
| `switchKey` | 是   | `string` | 灰度开关键。 |

## 响应字段

响应类型：`TResult`，源码里没有可展开的字段定义。
