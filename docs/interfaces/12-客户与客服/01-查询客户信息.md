# 查询客户信息

返回 [[12-客户与客服/00-模块总览|客户和客服接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                          |
| --------- | ------------------------------------------------------------- |
| 用途      | 查询当前用户绑定的客户编码和客户名称。                        |
| 源码方法  | `customerApi.queryCustomerInfo`                               |
| 源码位置  | `apps/com.deppon.app/src/services/customer/customer.api.ts:6` |
| HTTP 方法 | `POST`                                                        |
| 接口路径  | `/gwapi/userService/eco/user/secure/selectCustName`           |
| 登录态    | 默认是                                                        |
| 请求类型  | `无`                                                          |
| 响应类型  | `CustomerInfoRaw`                                             |

## 请求字段

无请求体。

## 响应字段

类型：`CustomerInfoRaw`。

| 字段            | 必填 | 类型           | 说明                         |
| --------------- | ---- | -------------- | ---------------------------- |
| `cusCode`       | 否   | string 或 null | 客户编码。                   |
| `custName`      | 否   | string 或 null | 客户名称。                   |
| `isMainLinkman` | 否   | string 或 null | 是否主联系人。               |
| `privateBill`   | 否   | string 或 null | 是否开启隐私面单或号码保护。 |
