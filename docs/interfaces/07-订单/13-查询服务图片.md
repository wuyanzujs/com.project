# 查询服务图片

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                                                                              |
| --------- | --------------------------------------------------------------------------------------------------------------------------------- |
| 用途      | 查询运单服务图片，快递和物流使用不同后端路径。                                                                                    |
| 源码方法  | `orderApi.queryServiceImages`                                                                                                     |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:212`                                                                         |
| HTTP 方法 | `POST`                                                                                                                            |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/queryLtlWayBillImages`<br>`/gwapi/waybillService/eco/wayBill/secure/queryWayBillImages` |
| 登录态    | 默认是                                                                                                                            |
| 请求类型  | `OrderServiceImagesRequest`                                                                                                       |
| 响应类型  | `string[]`                                                                                                                        |

## 请求字段

类型：`OrderServiceImagesRequest`。

| 字段         | 必填 | 类型     | 说明       |
| ------------ | ---- | -------- | ---------- |
| `wayBill`    | 是   | `string` | 运单号。   |
| `imageScene` | 是   | `string` | 图片场景。 |

## 响应字段

响应类型：`string[]`。
