# 查询电子合同状态

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 查询电子合同状态。                                           |
| 源码方法  | `orderApi.queryContractDetail`                               |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:283`    |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/onlineService/eco/online/secure/queryContractDetail` |
| 登录态    | 默认是                                                       |
| 请求类型  | `OrderContractDetailRequest`                                 |
| 响应类型  | `OrderContractDetailResponse`                                |

## 请求字段

类型：`OrderContractDetailRequest`。

| 字段            | 必填 | 类型     | 说明      |
| --------------- | ---- | -------- | --------- |
| `contractId`    | 是   | `string` | 合同 ID。 |
| `waybillNumber` | 是   | `string` | 运单号。  |

## 响应字段

类型：`OrderContractDetailResponse`。

| 字段     | 必填 | 类型     | 说明   |
| -------- | ---- | -------- | ------ |
| `status` | 是   | `string` | 状态。 |
