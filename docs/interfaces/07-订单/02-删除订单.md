# 删除订单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                      |
| --------- | --------------------------------------------------------- |
| 用途      | 删除订单列表中的订单记录。                                |
| 源码方法  | `orderApi.deleteOrder`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:110` |
| HTTP 方法 | `POST`                                                    |
| 接口路径  | `/gwapi/orderService/eco/order/secure/removeOrder`        |
| 登录态    | 默认是                                                    |
| 请求类型  | `OrderDeleteRequest`                                      |
| 响应类型  | `boolean`                                                 |

## 请求字段

类型：`OrderDeleteRequest`。

| 字段            | 必填 | 类型                      | 说明                                                |
| --------------- | ---- | ------------------------- | --------------------------------------------------- |
| `orderSource`   | 是   | 枚举："1"、"2"            | 订单来源。                                          |
| `orderListType` | 是   | 枚举："sender"、"receive" | 订单列表角色类型。                                  |
| `waybillNumber` | 否   | string 或 null            | 运单号。                                            |
| `orderNumber`   | 否   | string 或 null            | 订单号。                                            |
| `sysCode`       | 是   | `string`                  | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

响应类型：`boolean`。
