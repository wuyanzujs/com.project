# 提交催单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 提交催单请求。                                               |
| 源码方法  | `orderApi.submitUrge`                                        |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:137`    |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/orderService/eco/order/urgent/secure/v2/orderUrgent` |
| 登录态    | 默认是                                                       |
| 请求类型  | `OrderUrgeSubmitRequest`                                     |
| 响应类型  | `OrderUrgeSubmitResponse`                                    |

## 请求字段

类型：`OrderUrgeSubmitRequest`。

| 字段          | 必填 | 类型                                  | 说明             |
| ------------- | ---- | ------------------------------------- | ---------------- |
| `caseNo`      | 是   | `string`                              | 工单号或案件号。 |
| `reportScene` | 是   | 枚举："URGE_ORDER_NO"、"URGE_BILL_NO" | 上报场景。       |

## 响应字段

类型：`OrderUrgeSubmitResponse`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `errorMsg` | 否   | `string` | 错误信息。 |
