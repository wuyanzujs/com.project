# 查询回单图片

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                       |
| --------- | ---------------------------------------------------------- |
| 用途      | 查询签收回单图片和返单图片。                               |
| 源码方法  | `orderApi.queryReceiptImages`                              |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:180`  |
| HTTP 方法 | `POST`                                                     |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/signCounterfoil` |
| 登录态    | 默认是                                                     |
| 请求类型  | `OrderReceiptImagesRequest`                                |
| 响应类型  | `OrderReceiptImagesResponse`                               |

## 请求字段

类型：`OrderReceiptImagesRequest`。

| 字段      | 必填 | 类型     | 说明     |
| --------- | ---- | -------- | -------- |
| `wayBill` | 是   | `string` | 运单号。 |

## 响应字段

类型：`OrderReceiptImagesResponse`。

| 字段         | 必填 | 类型             | 说明           |
| ------------ | ---- | ---------------- | -------------- |
| `signList`   | 否   | string[] 或 null | 签收图片列表。 |
| `returnList` | 否   | string[] 或 null | 返单图片列表。 |
