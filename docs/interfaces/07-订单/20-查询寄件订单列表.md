# 查询寄件订单列表

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 查询寄件人视角的订单列表。                               |
| 源码方法  | `orderApi.querySenderList`                               |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:44` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/orderService/eco/order/secure/orderList`         |
| 登录态    | 默认是                                                   |
| 请求类型  | `SenderOrderListRequest`                                 |
| 响应类型  | `SenderOrderListResponse`                                |

## 请求字段

类型：`SenderOrderListRequest`。

| 字段                  | 必填 | 类型                                                                    | 说明                                                |
| --------------------- | ---- | ----------------------------------------------------------------------- | --------------------------------------------------- |
| `pageSize`            | 是   | `number`                                                                | 每页条数。                                          |
| `pageIndex`           | 是   | `number`                                                                | 页码，从当前业务传参看通常从 1 开始。               |
| `startTime`           | 是   | `string`                                                                | 开始时间。                                          |
| `endTime`             | 是   | `string`                                                                | 结束时间。                                          |
| `sysCode`             | 是   | `string`                                                                | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `ordersort`           | 是   | `0`                                                                     | 订单排序标识。                                      |
| `owsOrderListKeyword` | 否   | `string`                                                                | 订单列表搜索关键字。                                |
| `orderStatus`         | 是   | 枚举：空字符串、"INVALID"、"RECEIPTING"、"IN_TRANSIT"、"SIGN"、"CANCEL" | 订单状态。                                          |
| `paymentType`         | 是   | 枚举：空字符串、"MP"、"FC"、"CT"                                        | 付款方式。                                          |

## 响应字段

类型：`SenderOrderListResponse`。

| 字段             | 必填 | 类型                      | 说明           |
| ---------------- | ---- | ------------------------- | -------------- |
| `pageNum`        | 是   | `number`                  | 当前页码。     |
| `pageSize`       | 是   | `number`                  | 每页条数。     |
| `queryOrderList` | 是   | SenderOrderItem[] 或 null | 订单相关字段。 |
| `totalPage`      | 是   | `number`                  | 总页数。       |
| `totalRows`      | 是   | `number`                  | 总记录数。     |
| `weatherWarning` | 是   | string 或 null            | 天气预警提示。 |
