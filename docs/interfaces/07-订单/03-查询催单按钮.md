# 查询催单按钮

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                              |
| --------- | ----------------------------------------------------------------- |
| 用途      | 查询当前订单或运单是否可催单，以及可用催单按钮。                  |
| 源码方法  | `orderApi.queryUrgeButtons`                                       |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:117`         |
| HTTP 方法 | `POST`                                                            |
| 接口路径  | `/gwapi/onlineService/eco/online/complaint/secure/getUrgeButtons` |
| 登录态    | 默认是                                                            |
| 请求类型  | `OrderUrgeButtonRequest`                                          |
| 响应类型  | `OrderUrgeStatusResponse`                                         |

## 请求字段

类型：`OrderUrgeButtonRequest`。

| 字段        | 必填 | 类型                                  | 说明       |
| ----------- | ---- | ------------------------------------- | ---------- |
| `waybillNo` | 是   | `string`                              | 运单号。   |
| `urgeType`  | 是   | 枚举："URGE_ORDER_NO"、"URGE_BILL_NO" | 催单类型。 |

## 响应字段

类型：`OrderUrgeStatusResponse`。

| 字段         | 必填 | 类型                         | 说明       |
| ------------ | ---- | ---------------------------- | ---------- |
| `buttonList` | 否   | OrderUrgeButtonRaw[] 或 null | 按钮列表。 |
