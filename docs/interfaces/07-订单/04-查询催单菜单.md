# 查询催单菜单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 查询催单话术和催单菜单。                                     |
| 源码方法  | `orderApi.queryUrgeMenus`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:127`    |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/onlineService/eco/online/complaint/secure/urgeOrder` |
| 登录态    | 默认是                                                       |
| 请求类型  | `OrderUrgeButtonRequest`                                     |
| 响应类型  | `OrderUrgeMenusResponse`                                     |

## 请求字段

类型：`OrderUrgeButtonRequest`。

| 字段        | 必填 | 类型                                  | 说明       |
| ----------- | ---- | ------------------------------------- | ---------- |
| `waybillNo` | 是   | `string`                              | 运单号。   |
| `urgeType`  | 是   | 枚举："URGE_ORDER_NO"、"URGE_BILL_NO" | 催单类型。 |

## 响应字段

类型：`OrderUrgeMenusResponse`。

| 字段         | 必填 | 类型                         | 说明       |
| ------------ | ---- | ---------------------------- | ---------- |
| `speech`     | 否   | `string`                     | 催单话术。 |
| `buttonList` | 否   | OrderUrgeButtonRaw[] 或 null | 按钮列表。 |
