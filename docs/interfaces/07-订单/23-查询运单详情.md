# 查询运单详情

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                             |
| --------- | ---------------------------------------------------------------- |
| 用途      | 按运单号查询运单详情。                                           |
| 源码方法  | `orderApi.queryWaybillDetail`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:77`         |
| HTTP 方法 | `POST`                                                           |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/queryNewWaybillDetail` |
| 登录态    | 默认是                                                           |
| 请求类型  | `WaybillDetailRequest`                                           |
| 响应类型  | `WaybillDetailResponse`                                          |

## 请求字段

类型：`WaybillDetailRequest`。

| 字段            | 必填 | 类型     | 说明                                                |
| --------------- | ---- | -------- | --------------------------------------------------- |
| `waybillNumber` | 是   | `string` | 运单号。                                            |
| `sysCode`       | 是   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

类型：`WaybillDetailResponse`。

| 字段           | 必填 | 类型               | 说明           |
| -------------- | ---- | ------------------ | -------------- |
| `orderDetails` | 否   | `WaybillDetailRaw` | 订单相关字段。 |
