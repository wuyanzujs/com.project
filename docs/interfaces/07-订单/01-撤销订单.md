# 撤销订单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                      |
| --------- | --------------------------------------------------------- |
| 用途      | 撤销未完成订单。                                          |
| 源码方法  | `orderApi.cancelOrder`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:103` |
| HTTP 方法 | `POST`                                                    |
| 接口路径  | `/gwapi/orderService/eco/order/secure/revokeOrder`        |
| 登录态    | 默认是                                                    |
| 请求类型  | `OrderCancelRequest`                                      |
| 响应类型  | `boolean`                                                 |

## 请求字段

类型：`OrderCancelRequest`。

| 字段           | 必填 | 类型     | 说明                                                |
| -------------- | ---- | -------- | --------------------------------------------------- |
| `orderNumber`  | 是   | `string` | 订单号。                                            |
| `cancelReason` | 否   | `string` | 取消原因。                                          |
| `sysCode`      | 否   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

响应类型：`boolean`。
