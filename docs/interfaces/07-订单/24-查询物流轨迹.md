# 查询物流轨迹

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 查询运单轨迹列表。                                       |
| 源码方法  | `orderApi.queryTrackList`                                |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:87` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/trackService/eco/track/queryNewTrack`            |
| 登录态    | 默认是                                                   |
| 请求类型  | `{ waybillNumber: string }`                              |
| 响应类型  | `WaybillTrackListResponse`                               |

## 请求字段

类型：`{ waybillNumber: string }`。

| 字段            | 必填 | 类型     | 说明     |
| --------------- | ---- | -------- | -------- |
| `waybillNumber` | 是   | `string` | 运单号。 |

## 响应字段

类型：`WaybillTrackListResponse`。

| 字段             | 必填 | 类型                       | 说明           |
| ---------------- | ---- | -------------------------- | -------------- |
| `billNo`         | 是   | `string`                   | 运单号。       |
| `billNoState`    | 是   | `string`                   | 运单当前状态。 |
| `weatherWarning` | 是   | string 或 null             | 天气预警提示。 |
| `tracks`         | 是   | WaybillTrackItem[] 或 null | 物流轨迹列表。 |
