# 查询关注运单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目 | 说明 |
| --- | --- |
| 用途 | 查询当前账号已关注的运单列表。 |
| 源码方法 | `orderApi.querySubscriptions` |
| 源码位置 | `apps/com.deppon.app/src/services/order/order.api.ts:120` |
| HTTP 方法 | `GET` |
| 接口路径 | `/gwapi/waybillService/eco/wayBill/subscribe/list/records` |
| 登录态 | 默认是 |
| 请求类型 | 无 |
| 响应类型 | `Array<WaybillSubscriptionRaw \| null>` |

响应由 `normalizeWaybillSubscription` 归一为 `WaybillSubscriptionView`，页面不直接消费后端空值和角色字段。
