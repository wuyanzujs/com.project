# 查询收件订单列表

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                        |
| --------- | ----------------------------------------------------------- |
| 用途      | 查询收件人视角的订单列表。                                  |
| 源码方法  | `orderApi.queryConsigneeList`                               |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:54`    |
| HTTP 方法 | `POST`                                                      |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/receiveOrderList` |
| 登录态    | 默认是                                                      |
| 请求类型  | `ConsigneeOrderListRequest`                                 |
| 响应类型  | `ConsigneeOrderListResponse`                                |

## 请求字段

类型：`ConsigneeOrderListRequest`。

| 字段                  | 必填 | 类型                                                                    | 说明                                                |
| --------------------- | ---- | ----------------------------------------------------------------------- | --------------------------------------------------- |
| `type`                | 是   | `1`                                                                     | 业务类型。                                          |
| `status`              | 是   | `0`                                                                     | 状态。                                              |
| `pageSize`            | 是   | `number`                                                                | 每页条数。                                          |
| `pageIndex`           | 是   | `number`                                                                | 页码，从当前业务传参看通常从 1 开始。               |
| `startTime`           | 是   | `string`                                                                | 开始时间。                                          |
| `endTime`             | 是   | `string`                                                                | 结束时间。                                          |
| `sysCode`             | 是   | `string`                                                                | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `owsOrderListKeyword` | 否   | `string`                                                                | 订单列表搜索关键字。                                |
| `orderStatus`         | 是   | 枚举：空字符串、"INVALID"、"RECEIPTING"、"IN_TRANSIT"、"SIGN"、"CANCEL" | 订单状态。                                          |
| `paymentType`         | 是   | 枚举：空字符串、"MP"、"FC"、"CT"                                        | 付款方式。                                          |

## 响应字段

类型：`ConsigneeOrderListResponse`。

| 字段        | 必填 | 类型                         | 说明                                  |
| ----------- | ---- | ---------------------------- | ------------------------------------- |
| `total`     | 是   | `number`                     | 总记录数。                            |
| `pageSize`  | 是   | `number`                     | 每页条数。                            |
| `pageIndex` | 是   | `number`                     | 页码，从当前业务传参看通常从 1 开始。 |
| `row`       | 是   | ConsigneeOrderItem[] 或 null | 列表数据。                            |
