# 查询包装费用

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                      |
| --------- | --------------------------------------------------------- |
| 用途      | 查询包装费用明细。                                        |
| 源码方法  | `orderApi.queryPackagingFee`                              |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:273` |
| HTTP 方法 | `POST`                                                    |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/packagingFee`   |
| 登录态    | 默认是                                                    |
| 请求类型  | `OrderPackagingFeeRequest`                                |
| 响应类型  | `OrderPackagingFeeResponse`                               |

## 请求字段

类型：`OrderPackagingFeeRequest`。

| 字段            | 必填 | 类型     | 说明     |
| --------------- | ---- | -------- | -------- |
| `waybillNumber` | 是   | `string` | 运单号。 |

## 响应字段

类型：`OrderPackagingFeeResponse`。

| 字段                 | 必填 | 类型                               | 说明                   |
| -------------------- | ---- | ---------------------------------- | ---------------------- |
| `standVolume`        | 否   | number 或 null                     | 标准体积。             |
| `woodenFrameFee`     | 否   | number 或 null                     | 木架费用。             |
| `boxVolume`          | 否   | number 或 null                     | 木箱体积。             |
| `woodenCaseFee`      | 否   | number 或 null                     | 木箱费用。             |
| `salverNum`          | 否   | number 或 null                     | 托盘数量。             |
| `swoodenPalletFee`   | 否   | number 或 null                     | 标准木托盘费用。       |
| `nonSalverNum`       | 否   | number 或 null                     | 非托盘数量。           |
| `nwoodenPalletFee`   | 否   | number 或 null                     | 非标准木托盘费用。     |
| `waybillPackageList` | 否   | OrderPackagingFeeItemRaw[] 或 null | 运单包装费用明细列表。 |
