# 查询订单详情

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 按订单号查询订单详情。                                   |
| 源码方法  | `orderApi.queryOrderDetail`                              |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:67` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/orderService/eco/order/secure/orderDetail`       |
| 登录态    | 默认是                                                   |
| 请求类型  | `OrderDetailRequest`                                     |
| 响应类型  | `OrderDetail`                                            |

## 请求字段

类型：`OrderDetailRequest`。

| 字段          | 必填 | 类型     | 说明                                                |
| ------------- | ---- | -------- | --------------------------------------------------- |
| `orderNumber` | 是   | `string` | 订单号。                                            |
| `sysCode`     | 是   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
