# 查询包装图片

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 查询木架、木箱、托盘等包装图片。                             |
| 源码方法  | `orderApi.queryPackageImages`                                |
| 源码位置  | `apps/com.deppon.app/src/services/order/order.api.ts:264`    |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/waybillService/eco/wayBill/secure/getWoodPackagInfo` |
| 登录态    | 默认是                                                       |
| 请求类型  | `OrderPackageImagesRequest`                                  |
| 响应类型  | `OrderPackageImagesResponse`                                 |

## 请求字段

类型：`OrderPackageImagesRequest`。

| 字段      | 必填 | 类型     | 说明     |
| --------- | ---- | -------- | -------- |
| `wayBill` | 是   | `string` | 运单号。 |

## 响应字段

类型：`OrderPackageImagesResponse`。

| 字段   | 必填 | 类型                                     | 说明       |
| ------ | ---- | ---------------------------------------- | ---------- |
| `list` | 否   | { savePath?: string 或 null; }[] 或 null | 列表数据。 |
