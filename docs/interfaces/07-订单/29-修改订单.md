# 修改订单

返回 [[07-订单/00-模块总览|订单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目 | 说明 |
| --- | --- |
| 用途 | 修改后端允许编辑的待揽件订单基础信息。 |
| 源码方法 | `orderApi.modifyOrder` |
| 源码位置 | `apps/com.deppon.app/src/services/order/order.api.ts:113` |
| HTTP 方法 | `POST` |
| 接口路径 | `/gwapi/orderService/eco/order/secure/modifyOrder` |
| 登录态 | 默认是 |
| 请求类型 | `OrderModifyRequest` |
| 响应类型 | `boolean` |

## 请求字段

`orderNumber` 必填。其余字段均为可选差异字段：收寄件姓名、手机号、省市区、详细地址、货物名称、件数、重量、体积和备注。

当前 App 由 `buildOrderModifyRequest` 比较原始草稿和编辑草稿，只发送实际变化字段。产品、价格、付款、保价、代收货款、取件时间和增值服务修改后置，不能绕过对应报价、合同及扩展字段规则直接提交。
