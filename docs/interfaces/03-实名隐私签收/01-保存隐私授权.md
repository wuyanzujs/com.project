# 保存隐私授权

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 保存用户同意隐私协议的行为。                                 |
| 源码方法  | `privacyApi.savePrivacyVersion`                              |
| 源码位置  | `apps/com.deppon.app/src/services/privacy/privacy.api.ts:19` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/userService/eco/user/secure/userPrivacyBehavior`     |
| 登录态    | 默认是                                                       |
| 请求类型  | `PrivacyBehaviorRequest`                                     |
| 响应类型  | `boolean`                                                    |

## 请求字段

类型：`PrivacyBehaviorRequest`。

| 字段           | 必填 | 类型     | 说明                                                |
| -------------- | ---- | -------- | --------------------------------------------------- |
| `sysCode`      | 是   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `childSysCode` | 是   | `string` | 子系统来源编码。                                    |

## 响应字段

响应类型：`boolean`。
