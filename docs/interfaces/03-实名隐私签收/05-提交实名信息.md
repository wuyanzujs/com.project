# 提交实名信息

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 提交基础实名信息，使用姓名和身份证号完成实名。                 |
| 源码方法  | `realNameApi.submitRealNameAuth`                               |
| 源码位置  | `apps/com.deppon.app/src/services/realname/realname.api.ts:22` |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/userService/eco/user/secure/realNameAuth`              |
| 登录态    | 默认是                                                         |
| 请求类型  | `RealNameAuthRequest`                                          |
| 响应类型  | `boolean`                                                      |

## 请求字段

类型：`RealNameAuthRequest`。

| 字段         | 必填 | 类型       | 说明         |
| ------------ | ---- | ---------- | ------------ |
| `name`       | 是   | `string`   | 姓名或名称。 |
| `idCardNo`   | 是   | `string`   | 身份证号。   |
| `idCardType` | 是   | `"01"`     | 证件类型。   |
| `type`       | 是   | 枚举：0、2 | 业务类型。   |

## 响应字段

响应类型：`boolean`。
