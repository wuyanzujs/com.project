# 查询隐私协议版本

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                          |
| --------- | ------------------------------------------------------------- |
| 用途      | 查询用户已确认的隐私协议版本和当前最新版本。                  |
| 源码方法  | `privacyApi.queryPrivacyVersion`                              |
| 源码位置  | `apps/com.deppon.app/src/services/privacy/privacy.api.ts:9`   |
| HTTP 方法 | `POST`                                                        |
| 接口路径  | `/gwapi/userService/eco/user/secure/queryUserPrivacyBehavior` |
| 登录态    | 默认是                                                        |
| 请求类型  | `无`                                                          |
| 响应类型  | `PrivacyVersionResponse`                                      |

## 请求字段

无请求体。

## 响应字段

类型：`PrivacyVersionResponse`。

| 字段                | 必填 | 类型           | 说明                     |
| ------------------- | ---- | -------------- | ------------------------ |
| `lastVersionNumber` | 是   | string 或 null | 用户已确认的协议版本号。 |
| `versionNumber`     | 是   | string 或 null | 当前最新协议版本号。     |
