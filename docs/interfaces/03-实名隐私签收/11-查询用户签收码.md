# 查询用户签收码

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                         |
| ------- | -------------------------------------------------------------------------- |
| 用途      | 查询客户签收码，支持按定位经纬度获取附近签收码信息。                                                 |
| 源码方法    | `signApi.queryUserSignCode`                                                |
| 源码位置    | `apps/com.deppon.app/src/services/sign/sign.api.ts:25`                     |
| HTTP 方法 | `POST`                                                                     |
| 接口路径    | `/gwapi/onlineService/eco/online/courier/secure/queryCustomerSignCodeInfo` |
| 登录态     | 默认是                                                                        |
| 请求类型    | `UserSignCodeRequest`                                                      |
| 响应类型    | `UserSignCodeRaw`                                                          |

## 请求字段

类型：`UserSignCodeRequest`。

| 字段        | 必填 | 类型     | 说明   |
| ----------- | ---- | -------- | ------ |
| `latitude`  | 否   | `number` | 纬度。 |
| `longitude` | 否   | `number` | 经度。 |

## 响应字段

类型：`UserSignCodeRaw`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `signCode` | 是   | `string` | 签收码。   |
| `realName` | 是   | `string` | 真实姓名。 |
