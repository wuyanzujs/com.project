# 保存真实姓名

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 保存签收码场景使用的用户真实姓名。                       |
| 源码方法  | `signApi.saveUserRealName`                               |
| 源码位置  | `apps/com.deppon.app/src/services/sign/sign.api.ts:16`   |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/userService/eco/user/info/secure/updateRealName` |
| 登录态    | 默认是                                                   |
| 请求类型  | `SaveRealNameRequest`                                    |
| 响应类型  | `boolean`                                                |

## 请求字段

类型：`SaveRealNameRequest`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `realName` | 是   | `string` | 真实姓名。 |

## 响应字段

响应类型：`boolean`。
