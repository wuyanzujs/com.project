# 实名、隐私和签收码接口

返回 [[00-接口总览]]。

## 本模块接口

- [[03-实名隐私签收/01-保存隐私授权|保存隐私授权]]：保存用户同意隐私协议的行为。
- [[03-实名隐私签收/02-撤销隐私授权|撤销隐私授权]]：撤销用户隐私协议授权行为。
- [[03-实名隐私签收/03-查询隐私协议版本|查询隐私协议版本]]：查询用户已确认的隐私协议版本和当前最新版本。
- [[03-实名隐私签收/04-查询实名信息|查询实名信息]]：查询当前用户实名信息，用于实名状态展示。
- [[03-实名隐私签收/05-提交实名信息|提交实名信息]]：提交基础实名信息，使用姓名和身份证号完成实名。
- [[03-实名隐私签收/06-公安实名认证|公安实名认证]]：通过公安实名链路提交姓名和身份证号。
- [[03-实名隐私签收/07-网证实名认证|网证实名认证]]：通过网证链路提交实名授权数据。
- [[03-实名隐私签收/08-获取网证业务流水|获取网证业务流水]]：获取网证实名业务流水号。
- [[03-实名隐私签收/09-判断是否实名|判断是否实名]]：判断当前用户是否已经保存实名信息。
- [[03-实名隐私签收/10-保存真实姓名|保存真实姓名]]：保存签收码场景使用的用户真实姓名。
- [[03-实名隐私签收/11-查询用户签收码|查询用户签收码]]：查询客户签收码，支持按定位经纬度获取附近签收码信息。
