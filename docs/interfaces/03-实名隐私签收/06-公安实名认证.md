# 公安实名认证

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 通过公安实名链路提交姓名和身份证号。                           |
| 源码方法  | `realNameApi.submitPoliceRealName`                             |
| 源码位置  | `apps/com.deppon.app/src/services/realname/realname.api.ts:29` |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/userService/eco/user/idVerification/authByRealName`    |
| 登录态    | 默认是                                                         |
| 请求类型  | `PoliceRealNameRequest`                                        |
| 响应类型  | `boolean`                                                      |

## 请求字段

类型：`PoliceRealNameRequest`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `realName` | 是   | `string` | 真实姓名。 |
| `idCardNo` | 是   | `string` | 身份证号。 |

## 响应字段

响应类型：`boolean`。
