# 网证实名认证

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 通过网证链路提交实名授权数据。                                 |
| 源码方法  | `realNameApi.submitNetworkIdAuth`                              |
| 源码位置  | `apps/com.deppon.app/src/services/realname/realname.api.ts:36` |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/userService/eco/user/idVerification/authByNetworkId`   |
| 登录态    | 默认是                                                         |
| 请求类型  | `NetworkIdAuthRequest`                                         |
| 响应类型  | `boolean`                                                      |

## 请求字段

类型：`NetworkIdAuthRequest`。

| 字段             | 必填 | 类型     | 说明                   |
| ---------------- | ---- | -------- | ---------------------- |
| `bizSeq`         | 是   | `string` | 实名或网证业务流水号。 |
| `idCardAuthData` | 是   | `string` | 网证实名授权数据。     |

## 响应字段

响应类型：`boolean`。
