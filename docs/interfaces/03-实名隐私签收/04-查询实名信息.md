# 查询实名信息

返回 [[03-实名隐私签收/00-模块总览|实名、隐私和签收码接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 查询当前用户实名信息，用于实名状态展示。                       |
| 源码方法  | `realNameApi.queryRealNameAuth`                                |
| 源码位置  | `apps/com.deppon.app/src/services/realname/realname.api.ts:11` |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/userService/eco/user/secure/queryRealNameAuth`         |
| 登录态    | 默认是                                                         |
| 请求类型  | `无`                                                           |
| 响应类型  | `RealNameAuthRaw`                                              |

## 请求字段

无请求体。

## 响应字段

类型：`RealNameAuthRaw`。

| 字段       | 必填 | 类型     | 说明         |
| ---------- | ---- | -------- | ------------ |
| `name`     | 否   | `string` | 姓名或名称。 |
| `idCardNo` | 否   | `string` | 身份证号。   |
