# 查询优惠券列表

返回 [[10-优惠券/00-模块总览|优惠券接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                            |
| --------- | --------------------------------------------------------------- |
| 用途      | 按优惠券状态查询用户优惠券列表。                                |
| 源码方法  | `couponApi.queryUserCouponList`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/coupon/coupon.api.ts:13`      |
| HTTP 方法 | `POST`                                                          |
| 接口路径  | `/gwapi/couponService/eco/coupon/new/secure/queryNewCouponList` |
| 登录态    | 默认是                                                          |
| 请求类型  | `UserCouponListRequest`                                         |
| 响应类型  | `CouponItem[]`                                                  |

## 请求字段

类型：`UserCouponListRequest`。

| 字段   | 必填 | 类型                                                           | 说明       |
| ------ | ---- | -------------------------------------------------------------- | ---------- |
| `type` | 是   | 枚举："AVAILABLE"、"UN_AVAILABLE"、"USABLE"、"USED"、"EXPIRED" | 业务类型。 |

## 响应字段

类型：`CouponItem[]`，数组元素字段如下。

| 字段                 | 必填 | 类型           | 说明           |
| -------------------- | ---- | -------------- | -------------- |
| `couponCode`         | 是   | `string`       | 优惠券编码。   |
| `startTime`          | 是   | `string`       | 开始时间。     |
| `endTime`            | 是   | `string`       | 结束时间。     |
| `discountFee`        | 是   | `number`       | 优惠金额。     |
| `useFee`             | 是   | `number`       | 使用门槛金额。 |
| `subFee`             | 是   | `number`       | 减免金额。     |
| `limitDiscountFee`   | 是   | `number`       | 最高优惠金额。 |
| `subtitle`           | 否   | `string`       | 副标题。       |
| `subTitle`           | 否   | `string`       | 副标题。       |
| `discountType`       | 是   | `string`       | 优惠类型。     |
| `fitWeek`            | 否   | string 或 null | 适用星期。     |
| `fitChannel`         | 否   | string 或 null | 适用渠道。     |
| `label`              | 否   | `string`       | 展示标签。     |
| `optimalFee`         | 否   | `number`       | 最优优惠金额。 |
| `useLimit`           | 否   | `string[]`     | 使用限制。     |
| `status`             | 否   | `string`       | 状态。         |
| `promotionReduceFee` | 否   | `number`       | 活动减免金额。 |
| `couponLabel`        | 否   | `string`       | 优惠券标签。   |
| `employeeFlag`       | 否   | 枚举："Y"、"N" | 是否员工券。   |
| `businessStatus`     | 否   | `string`       | 业务状态。     |
| `fitCouponTagList`   | 否   | `string[]`     | 适用标签列表。 |
