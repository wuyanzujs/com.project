# 兑换优惠券

返回 [[10-优惠券/00-模块总览|优惠券接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 使用兑换码兑换优惠券。                                         |
| 源码方法  | `couponApi.exchangeCoupon`                                     |
| 源码位置  | `apps/com.deppon.app/src/services/coupon/coupon.api.ts:23`     |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/couponService/eco/coupon/coupon/secure/exchangeCoupon` |
| 登录态    | 默认是                                                         |
| 请求类型  | `CouponExchangeRequest`                                        |
| 响应类型  | `CouponExchangeResponse`                                       |

## 请求字段

类型：`CouponExchangeRequest`。

| 字段                 | 必填 | 类型     | 说明           |
| -------------------- | ---- | -------- | -------------- |
| `exchangeCouponCode` | 是   | `string` | 优惠券兑换码。 |

## 响应字段

类型：`CouponExchangeResponse`。

| 字段   | 必填 | 类型      | 说明                 |
| ------ | ---- | --------- | -------------------- |
| `data` | 否   | `boolean` | 业务数据。           |
| `msg`  | 否   | `string`  | 后端返回的提示消息。 |
