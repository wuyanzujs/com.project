# 查询优惠券详情

返回 [[10-优惠券/00-模块总览|优惠券接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 查询优惠券详情、适用产品、限制说明和地址限制。                 |
| 源码方法  | `couponApi.queryCouponDetail`                                  |
| 源码位置  | `apps/com.deppon.app/src/services/coupon/coupon.api.ts:35`     |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/couponService/eco/coupon/new/secure/queryCouponDetail` |
| 登录态    | 默认是                                                         |
| 请求类型  | `CouponDetailRequest`                                          |
| 响应类型  | `CouponDetailRaw`                                              |

## 请求字段

类型：`CouponDetailRequest`。

| 字段         | 必填 | 类型     | 说明         |
| ------------ | ---- | -------- | ------------ |
| `couponCode` | 是   | `string` | 优惠券编码。 |

## 响应字段

类型：`CouponDetailRaw`。

| 字段             | 必填 | 类型              | 说明           |
| ---------------- | ---- | ----------------- | -------------- |
| `couponCode`     | 是   | `string`          | 优惠券编码。   |
| `fitProduct`     | 否   | string 或 null    | 适用产品。     |
| `limit`          | 否   | string[] 或 null  | 限制条件。     |
| `couponDescribe` | 否   | string 或 null    | 优惠券说明。   |
| `addressList`    | 否   | `CouponAddress[]` | 适用地址列表。 |
