# 生成 E 卡预览链接

返回 [[11-会员与E卡/00-模块总览|会员与 E 卡接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                 |
| --------- | -------------------------------------------------------------------- |
| 用途      | 未登录场景生成 E 卡访客预览页链接。                                  |
| 源码方法  | `ecardApi.createECardPreviewLink`                                    |
| 源码位置  | `apps/com.deppon.app/src/services/ecard/ecard.api.ts:21`             |
| HTTP 方法 | `POST`                                                               |
| 接口路径  | `/gwapi/onlineService/eco/online/prestore/getPmcPreStoreVisitorPage` |
| 登录态    | 否                                                                   |
| 请求类型  | `Pick<ECardLinkRequest, 'sysCode'>`                                  |
| 响应类型  | `ECardLinkResponse`                                                  |

## 请求字段

类型：`Pick<ECardLinkRequest, 'sysCode'>`。

| 字段      | 必填 | 类型     | 说明                                                |
| --------- | ---- | -------- | --------------------------------------------------- |
| `sysCode` | 是   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

类型：`ECardLinkResponse`。

| 字段    | 必填 | 类型           | 说明                           |
| ------- | ---- | -------------- | ------------------------------ |
| `form`  | 是   | `string`       | 后端返回的表单或跳转表单内容。 |
| `payNo` | 是   | string 或 null | 支付单号。                     |
