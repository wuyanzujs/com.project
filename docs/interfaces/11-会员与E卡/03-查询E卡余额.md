# 查询 E 卡余额

返回 [[11-会员与E卡/00-模块总览|会员与 E 卡接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 查询 E 卡余额和开户状态。                                |
| 源码方法  | `ecardApi.queryBalance`                                  |
| 源码位置  | `apps/com.deppon.app/src/services/ecard/ecard.api.ts:32` |
| HTTP 方法 | `GET`                                                    |
| 接口路径  | `/gwapi/memberService/eco/member/secure/getEcardBalance` |
| 登录态    | 否                                                       |
| 请求类型  | `无`                                                     |
| 响应类型  | `ECardBalanceRaw`                                        |

## 请求字段

无请求体。

## 响应字段

类型：`ECardBalanceRaw`。

| 字段            | 必填 | 类型      | 说明             |
| --------------- | ---- | --------- | ---------------- |
| `balance`       | 是   | `string`  | 余额。           |
| `existCustomer` | 是   | `boolean` | 是否已有客户。   |
| `passwordSet`   | 是   | `boolean` | 是否已设置密码。 |
| `rechargeDesc`  | 是   | `string`  | 充值说明。       |
