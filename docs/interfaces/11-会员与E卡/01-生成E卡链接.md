# 生成 E 卡链接

返回 [[11-会员与E卡/00-模块总览|会员与 E 卡接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                 |
| --------- | -------------------------------------------------------------------- |
| 用途      | 生成 E 卡中心、账单或充值页面链接。                                  |
| 源码方法  | `ecardApi.createECardLink`                                           |
| 源码位置  | `apps/com.deppon.app/src/services/ecard/ecard.api.ts:11`             |
| HTTP 方法 | `POST`                                                               |
| 接口路径  | `/gwapi/onlineService/eco/online/prestore/secure/getPmcPrestorePage` |
| 登录态    | 默认是                                                               |
| 请求类型  | `ECardLinkRequest`                                                   |
| 响应类型  | `ECardLinkResponse`                                                  |

## 请求字段

类型：`ECardLinkRequest`。

| 字段           | 必填 | 类型              | 说明                                                |
| -------------- | ---- | ----------------- | --------------------------------------------------- |
| `sysCode`      | 是   | `string`          | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `type`         | 否   | `ECardType`       | 业务类型。                                          |
| `activityCode` | 否   | `string`          | 活动编码。                                          |
| `targetPage`   | 否   | `ECardTargetPage` | 目标页面。                                          |
| `source`       | 否   | `string`          | 来源标识。                                          |
| `postmanId`    | 否   | `string`          | 快递员 ID。                                         |

## 响应字段

类型：`ECardLinkResponse`。

| 字段    | 必填 | 类型           | 说明                           |
| ------- | ---- | -------------- | ------------------------------ |
| `form`  | 是   | `string`       | 后端返回的表单或跳转表单内容。 |
| `payNo` | 是   | string 或 null | 支付单号。                     |
