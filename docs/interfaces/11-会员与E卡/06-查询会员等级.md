# 查询会员等级

返回 [[11-会员与E卡/00-模块总览|会员与 E 卡接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                 |
| --------- | -------------------------------------------------------------------- |
| 用途      | 查询会员等级和成长值。                                               |
| 源码方法  | `memberApi.queryLevel`                                               |
| 源码位置  | `apps/com.deppon.app/src/services/member/member.api.ts:17`           |
| HTTP 方法 | `GET`                                                                |
| 接口路径  | `/gwapi/memberService/eco/member/grade/secure/weChatMiniMemberGrade` |
| 登录态    | 否                                                                   |
| 请求类型  | `无`                                                                 |
| 响应类型  | `MemberLevelRaw`                                                     |

## 请求字段

无请求体。

## 响应字段

类型：`MemberLevelRaw`。

| 字段             | 必填 | 类型     | 说明                 |
| ---------------- | ---- | -------- | -------------------- |
| `levelName`      | 否   | `string` | 会员等级名称。       |
| `levelCode`      | 否   | `number` | 会员等级编码。       |
| `growthValue`    | 否   | `number` | 当前成长值。         |
| `maxGrowthValue` | 否   | `number` | 当前等级最大成长值。 |
