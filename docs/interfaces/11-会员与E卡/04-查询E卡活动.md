# 查询 E 卡活动

返回 [[11-会员与E卡/00-模块总览|会员与 E 卡接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                                      |
| --------- | ------------------------------------------------------------------------- |
| 用途      | 查询 E 卡充值活动配置。                                                   |
| 源码方法  | `ecardApi.queryPromotions`                                                |
| 源码位置  | `apps/com.deppon.app/src/services/ecard/ecard.api.ts:45`                  |
| HTTP 方法 | `POST`                                                                    |
| 接口路径  | `/gwapi/onlineService/eco/online/prestore/secure/queryRechargePromotions` |
| 登录态    | 否                                                                        |
| 请求类型  | `无`                                                                      |
| 响应类型  | `ECardPromotionRaw`                                                       |

## 请求字段

无请求体。

## 响应字段

类型：`ECardPromotionRaw`。

| 字段         | 必填 | 类型                                                              | 说明       |
| ------------ | ---- | ----------------------------------------------------------------- | ---------- |
| `configList` | 否   | `{ rechargeAmountDesc?: string; discountAmountDesc?: string; }[]` | 配置列表。 |
