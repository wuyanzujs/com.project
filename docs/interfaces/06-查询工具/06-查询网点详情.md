# 查询网点详情

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                       |
| --------- | ---------------------------------------------------------- |
| 用途      | 查询网点详情。                                             |
| 源码方法  | `queryApi.queryStationDetail`                              |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:69`   |
| HTTP 方法 | `POST`                                                     |
| 接口路径  | `/gwapi/queryService/eco/query/branch/queryDeptInfoByCode` |
| 登录态    | 否                                                         |
| 请求类型  | `StationDetailRequest`                                     |
| 响应类型  | `StationDetailRaw`                                         |

## 请求字段

类型：`StationDetailRequest`。

| 字段       | 必填 | 类型     | 说明             |
| ---------- | ---- | -------- | ---------------- |
| `deptCode` | 是   | `string` | 部门或网点编码。 |

## 响应字段

类型：`StationDetailRaw`。

| 字段            | 必填 | 类型           | 说明             |
| --------------- | ---- | -------------- | ---------------- |
| `code`          | 否   | string 或 null | 编码。           |
| `deptCode`      | 否   | string 或 null | 部门或网点编码。 |
| `unifiedCode`   | 否   | string 或 null | 编码字段。       |
| `name`          | 否   | string 或 null | 姓名或名称。     |
| `deptName`      | 否   | string 或 null | 部门或网点名称。 |
| `address`       | 否   | string 或 null | 详细地址。       |
| `deptAddress`   | 否   | string 或 null | 地址字段。       |
| `contactway`    | 否   | string 或 null | 联系电话。       |
| `contactWay`    | 否   | string 或 null | 联系电话。       |
| `businessType`  | 否   | string 或 null | 业务类型。       |
| `businessScope` | 否   | string 或 null | 业务范围。       |
| `longitude`     | 否   | string 或 null | 经度。           |
| `latitude`      | 否   | string 或 null | 纬度。           |
| `baiduLng`      | 否   | string 或 null | 百度坐标经度。   |
| `baiduLat`      | 否   | string 或 null | 百度坐标纬度。   |
| `startTime`     | 否   | string 或 null | 开始时间。       |
| `endTime`       | 否   | string 或 null | 结束时间。       |
