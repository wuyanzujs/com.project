# 查询派送街道

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 查询地址对应的可派送街道或乡镇信息。                     |
| 源码方法  | `queryApi.queryDispatchStreet`                           |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:17` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/queryService/eco/query/addressImagineTown`       |
| 登录态    | 否                                                       |
| 请求类型  | `DispatchStreetRequest`                                  |
| 响应类型  | `string`                                                 |

## 请求字段

类型：`DispatchStreetRequest`。

| 字段       | 必填 | 类型     | 说明       |
| ---------- | ---- | -------- | ---------- |
| `province` | 是   | `string` | 省份名称。 |
| `city`     | 是   | `string` | 城市名称。 |
| `county`   | 是   | `string` | 区县名称。 |
| `address`  | 是   | `string` | 详细地址。 |

## 响应字段

响应类型：`string`。
