# 查询物流派送范围

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                          |
| --------- | ------------------------------------------------------------- |
| 用途      | 查询物流派送范围。                                            |
| 源码方法  | `queryApi.queryLogisticsDispatch`                             |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:38`      |
| HTTP 方法 | `GET`                                                         |
| 接口路径  | `/gwapi/queryService/eco/query/logisticsRange/queryAreaRange` |
| 登录态    | 否                                                            |
| 请求类型  | `{ code: string }`                                            |
| 响应类型  | `LogisticsDispatchResponse`                                   |

## 请求字段

类型：`{ code: string }`。

| 字段   | 必填 | 类型     | 说明   |
| ------ | ---- | -------- | ------ |
| `code` | 是   | `string` | 编码。 |

## 响应字段

类型：`LogisticsDispatchResponse`。

| 字段                    | 必填 | 类型                            | 说明       |
| ----------------------- | ---- | ------------------------------- | ---------- |
| `noDeliverList`         | 是   | LogisticsDispatchItem[] 或 null | 列表字段。 |
| `normalFarDeliveryList` | 是   | LogisticsDispatchItem[] 或 null | 列表字段。 |
| `specialDeliverList`    | 是   | LogisticsDispatchItem[] 或 null | 列表字段。 |
