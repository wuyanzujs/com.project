# 查询快递派送范围

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                      |
| --------- | --------------------------------------------------------- |
| 用途      | 查询快递派送范围。                                        |
| 源码方法  | `queryApi.queryExpressDispatch`                           |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:28`  |
| HTTP 方法 | `POST`                                                    |
| 接口路径  | `/gwapi/queryService/eco/query/range/queryAddressImagine` |
| 登录态    | 否                                                        |
| 请求类型  | `ExpressDispatchRequest`                                  |
| 响应类型  | `ExpressDispatchResponse`                                 |

## 请求字段

类型：`ExpressDispatchRequest`。

| 字段         | 必填 | 类型     | 说明       |
| ------------ | ---- | -------- | ---------- |
| `cityCode`   | 是   | `string` | 城市编码。 |
| `countyCode` | 是   | `string` | 区县编码。 |

## 响应字段

类型：`ExpressDispatchResponse`。

| 字段           | 必填 | 类型                          | 说明       |
| -------------- | ---- | ----------------------------- | ---------- |
| `deliveryList` | 是   | ExpressDispatchItem[] 或 null | 列表字段。 |
