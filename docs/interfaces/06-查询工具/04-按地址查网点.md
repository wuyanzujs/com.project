# 按地址查网点

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 按具体地址查询附近网点。                                 |
| 源码方法  | `queryApi.queryAddressStations`                          |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:49` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/queryService/eco/query/branch/stationSearch`     |
| 登录态    | 否                                                       |
| 请求类型  | `AddressStationRequest`                                  |
| 响应类型  | `AddressStationResponse`                                 |

## 请求字段

类型：`AddressStationRequest`。

| 字段           | 必填 | 类型            | 说明                 |
| -------------- | ---- | --------------- | -------------------- |
| `province`     | 是   | `string`        | 省份名称。           |
| `city`         | 是   | `string`        | 城市名称。           |
| `county`       | 是   | `string`        | 区县名称。           |
| `otherAddress` | 是   | `string`        | 补充地址或详细地址。 |
| `matchtypes`   | 是   | `string[]`      | 网点匹配类型列表。   |
| `transportway` | 是   | `string`        | 运输方式。           |
| `lat`          | 否   | `number`        | 纬度。               |
| `lng`          | 否   | `number`        | 经度。               |
| `fiveKmType`   | 否   | 枚举："1"、null | 5 公里范围查询类型。 |

## 响应字段

类型：`AddressStationResponse`。

| 字段   | 必填 | 类型                            | 说明       |
| ------ | ---- | ------------------------------- | ---------- |
| `data` | 是   | AddressStationRawItem[] 或 null | 业务数据。 |
