# 按城市查网点

返回 [[06-查询工具/00-模块总览|查询工具接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                     |
| --------- | -------------------------------------------------------- |
| 用途      | 按城市行政区分页查询网点。                               |
| 源码方法  | `queryApi.queryCityStations`                             |
| 源码位置  | `apps/com.deppon.app/src/services/query/query.api.ts:59` |
| HTTP 方法 | `POST`                                                   |
| 接口路径  | `/gwapi/queryService/eco/query/branch/queryDeptNet`      |
| 登录态    | 否                                                       |
| 请求类型  | `CityStationRequest`                                     |
| 响应类型  | `CityStationResponse`                                    |

## 请求字段

类型：`CityStationRequest`。

| 字段           | 必填 | 类型                               | 说明                                  |
| -------------- | ---- | ---------------------------------- | ------------------------------------- |
| `level`        | 是   | `"4"`                              | 行政区划层级，当前固定为区县级。      |
| `type`         | 是   | 枚举：空字符串、"1"、"4"、"2"、"3" | 业务类型。                            |
| `provinceCode` | 是   | `string`                           | 省份编码。                            |
| `provinceName` | 是   | `string`                           | 省份名称。                            |
| `cityCode`     | 是   | `string`                           | 城市编码。                            |
| `cityName`     | 是   | `string`                           | 城市名称。                            |
| `countyCode`   | 是   | `string`                           | 区县编码。                            |
| `countyName`   | 是   | `string`                           | 区县名称。                            |
| `pageIndex`    | 是   | `number`                           | 页码，从当前业务传参看通常从 1 开始。 |
| `pageSize`     | 是   | `number`                           | 每页条数。                            |

## 响应字段

类型：`CityStationResponse`。

| 字段           | 必填 | 类型                         | 说明       |
| -------------- | ---- | ---------------------------- | ---------- |
| `totalRows`    | 是   | number 或 null               | 总记录数。 |
| `netResponses` | 是   | CityStationRawItem[] 或 null | 网点列表。 |
