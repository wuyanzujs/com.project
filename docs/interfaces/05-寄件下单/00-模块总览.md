# 寄件下单接口

返回 [[00-接口总览]]。

## 本模块接口

- [[05-寄件下单/01-创建寄件订单|创建寄件订单]]：创建寄件订单，提交寄件人、收件人、货物、服务、预约等完整下单参数。
- [[05-寄件下单/02-查询寄件订单详情|查询寄件订单详情]]：寄件模块内查询订单详情，用于下单结果或详情回显。
- [[05-寄件下单/03-取消寄件订单|取消寄件订单]]：寄件模块内撤销订单。
- [[05-寄件下单/04-查询价格时效|查询价格时效]]：查询价格时效，返回可用产品、预计到达时间、费用明细和计费重量。
- [[05-寄件下单/05-查询预约取件时间|查询预约取件时间]]：查询可预约上门时间和揽收网点。
- [[05-寄件下单/06-查询货物品名|查询货物品名]]：根据关键字查询货物品名候选。
- [[05-寄件下单/07-查询货物标签|查询货物标签]]：查询货物品名风险标签，例如禁寄、易碎、高价值等。
- [[05-寄件下单/08-查询保价费用|查询保价费用]]：查询保价或定额保障费用。
- [[05-寄件下单/09-下单前拦截校验|下单前拦截校验]]：下单前过滤校验，判断收寄件、货物、产品组合是否命中拦截规则。
- [[05-寄件下单/10-检查是否可下单|检查是否可下单]]：检查当前用户是否允许继续下单。
