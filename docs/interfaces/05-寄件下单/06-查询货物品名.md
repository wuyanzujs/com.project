# 查询货物品名

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 根据关键字查询货物品名候选。                                 |
| 源码方法  | `expressApi.queryGoodsName`                                  |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:47` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/onlineService/eco/online/cargo/queryCargoInfo`       |
| 登录态    | 默认是                                                       |
| 请求类型  | `ExpressGoodsNameRequest`                                    |
| 响应类型  | `ExpressGoodsNameResponse`                                   |

## 请求字段

类型：`ExpressGoodsNameRequest`。

| 字段        | 必填 | 类型     | 说明                                  |
| ----------- | ---- | -------- | ------------------------------------- |
| `keyWord`   | 是   | `string` | 搜索关键字。                          |
| `pageIndex` | 是   | `number` | 页码，从当前业务传参看通常从 1 开始。 |
| `pageSize`  | 是   | `number` | 每页条数。                            |

## 响应字段

类型：`ExpressGoodsNameResponse`。

| 字段        | 必填 | 类型                       | 说明       |
| ----------- | ---- | -------------------------- | ---------- |
| `pageSize`  | 是   | `number`                   | 每页条数。 |
| `pageNum`   | 是   | `number`                   | 当前页码。 |
| `totalPage` | 是   | `number`                   | 总页数。   |
| `totalRows` | 是   | `number`                   | 总记录数。 |
| `list`      | 是   | ExpressGoodsItem[] 或 null | 列表数据。 |
