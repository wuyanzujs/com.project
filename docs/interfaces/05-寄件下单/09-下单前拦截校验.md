# 下单前拦截校验

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 下单前过滤校验，判断收寄件、货物、产品组合是否命中拦截规则。 |
| 源码方法  | `expressApi.filterOrder`                                     |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:83` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/orderService/eco/order/sieveOrder/tips`              |
| 登录态    | 默认是                                                       |
| 请求类型  | `ExpressFilterRequest`                                       |
| 响应类型  | `ExpressFilterResponse`                                      |

## 请求字段

类型：`ExpressFilterRequest`。

| 字段              | 必填 | 类型                                                                                                           | 说明                     |
| ----------------- | ---- | -------------------------------------------------------------------------------------------------------------- | ------------------------ |
| `contactAddress`  | 是   | `string`                                                                                                       | 寄件详细地址。           |
| `contactMobile`   | 是   | `string`                                                                                                       | 寄件人手机号。           |
| `contactName`     | 是   | `string`                                                                                                       | 寄件人姓名。             |
| `receiverAddress` | 是   | `string`                                                                                                       | 收件详细地址。           |
| `receiverName`    | 是   | `string`                                                                                                       | 收件人姓名。             |
| `receiverMobile`  | 是   | `string`                                                                                                       | 收件人手机号。           |
| `goodsName`       | 是   | `string`                                                                                                       | 货物名称。               |
| `totalWeight`     | 是   | `number`                                                                                                       | 总重量。                 |
| `transportMode`   | 是   | 枚举：空字符串、"PACKAGE"、"DEAP"、"RCP"、"PCP"、"DCZP"、"DJBK"、"DJTK"、"XJBK"、"XJTK"、"XJTH"、"DJTH"、"YTY" | 运输产品或产品编码。     |
| `deliveryMode`    | 是   | 枚举：空字符串、"PICKNOTUPSTAIRS"、"PICKSELF"、"PICKUPSTAIRS"、"BIGUPSTAIRS"                                   | 派送方式。               |
| `customerCode`    | 否   | `string`                                                                                                       | 客户编码。               |
| `limitCust`       | 否   | 枚举：0、1                                                                                                     | 是否限制客户下单的标识。 |

## 响应字段

类型：`ExpressFilterResponse`。

| 字段        | 必填 | 类型                | 说明             |
| ----------- | ---- | ------------------- | ---------------- |
| `reason`    | 是   | `string`            | 原因说明。       |
| `type`      | 是   | 枚举：0、1、2、3、4 | 业务类型。       |
| `depotType` | 否   | `string`            | 网点或仓库类型。 |
