# 查询寄件订单详情

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                          |
| --------- | ------------------------------------------------------------- |
| 用途      | 寄件模块内查询订单详情，用于下单结果或详情回显。              |
| 源码方法  | `expressApi.queryOrderDetail`                                 |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:118` |
| HTTP 方法 | `POST`                                                        |
| 接口路径  | `/gwapi/orderService/eco/order/secure/orderDetail`            |
| 登录态    | 默认是                                                        |
| 请求类型  | `ExpressOrderDetailRequest`                                   |
| 响应类型  | `ExpressOrderDetail`                                          |

## 请求字段

类型：`ExpressOrderDetailRequest`。

| 字段            | 必填 | 类型     | 说明                                                |
| --------------- | ---- | -------- | --------------------------------------------------- |
| `orderNumber`   | 否   | `string` | 订单号。                                            |
| `waybillNumber` | 否   | `string` | 运单号。                                            |
| `sysCode`       | 否   | `string` | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |

## 响应字段

类型：`ExpressOrderDetail`。

| 字段            | 必填 | 类型     | 说明         |
| --------------- | ---- | -------- | ------------ |
| `orderNumber`   | 否   | `string` | 订单号。     |
| `waybillNumber` | 否   | `string` | 运单号。     |
| `orderStatus`   | 否   | `string` | 订单状态。   |
| `contactName`   | 否   | `string` | 寄件人姓名。 |
| `receiverName`  | 否   | `string` | 收件人姓名。 |

说明：该类型带有索引签名，后端可能返回额外字段，当前文档只展开源码中已声明的字段。
