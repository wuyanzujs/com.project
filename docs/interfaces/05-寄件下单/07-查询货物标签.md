# 查询货物标签

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                         |
| --------- | ------------------------------------------------------------ |
| 用途      | 查询货物品名风险标签，例如禁寄、易碎、高价值等。             |
| 源码方法  | `expressApi.queryGoodsLabels`                                |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:58` |
| HTTP 方法 | `POST`                                                       |
| 接口路径  | `/gwapi/orderService/eco/order/queryGoodsRemark`             |
| 登录态    | 默认是                                                       |
| 请求类型  | `ExpressGoodsLabelRequest`                                   |
| 响应类型  | `ExpressGoodsLabel[]`                                        |

## 请求字段

类型：`ExpressGoodsLabelRequest`。

| 字段                 | 必填 | 类型     | 说明       |
| -------------------- | ---- | -------- | ---------- |
| `goodsName`          | 是   | `string` | 货物名称。 |
| `senderProvinceName` | 否   | `string` | 名称字段。 |
| `senderCityName`     | 否   | `string` | 名称字段。 |
| `senderCountyName`   | 否   | `string` | 名称字段。 |
| `arriveProvinceName` | 否   | `string` | 名称字段。 |
| `arriveCityName`     | 否   | `string` | 名称字段。 |
| `arriveCountyName`   | 否   | `string` | 名称字段。 |

## 响应字段

类型：`ExpressGoodsLabel[]`，数组元素字段如下。

| 字段              | 必填 | 类型                                                                                                                                                                                                                     | 说明                                           |
| ----------------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------- |
| `tip`             | 是   | `string`                                                                                                                                                                                                                 | 提示文案。                                     |
| `goodsRemarkCode` | 是   | 枚举：空字符串、"other_category"、"3c_category"、"fragile_articles"、"high_value_category"、"fresh_category"、"unknow_category"、"contraband_category"、"battery_category"、"limitation_insure"、"worry_free_protection" | 编码字段。                                     |
| `displayType`     | 是   | 枚举："alert"、"tips"、"addprice"、"forbid"、null                                                                                                                                                                        | 前端展示方式，例如弹窗、提示、加价或禁止寄递。 |
