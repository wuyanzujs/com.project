# 创建寄件订单

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                               |
| --------- | ------------------------------------------------------------------ |
| 用途      | 创建寄件订单，提交寄件人、收件人、货物、服务、预约等完整下单参数。 |
| 源码方法  | `expressApi.createOrder`                                           |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:105`      |
| HTTP 方法 | `POST`                                                             |
| 接口路径  | `/gwapi/orderService/eco/order/mysql/createOrder`                  |
| 登录态    | 默认是                                                             |
| 请求类型  | `CreateExpressOrderRequest`                                        |
| 响应类型  | `CreateExpressOrderResponse`                                       |

## 请求字段

类型：`CreateExpressOrderRequest`。

| 字段                   | 必填 | 类型                    | 说明                     |
| ---------------------- | ---- | ----------------------- | ------------------------ |
| `contactIdList`        | 是   | `string[]`              | 联系人 ID 列表。         |
| `isAgreement`          | 否   | `ExpressFlag`           | 是否同意协议。           |
| `isContact`            | 是   | 枚举："Y"、"N"          | 是否联系寄件人。         |
| `clientChannel`        | 是   | `string`                | 客户端渠道。             |
| `contactName`          | 是   | `string`                | 寄件人姓名。             |
| `contactMobile`        | 是   | `string`                | 寄件人手机号。           |
| `contactProvince`      | 是   | `string`                | 寄件省份。               |
| `contactCity`          | 是   | `string`                | 寄件城市。               |
| `contactArea`          | 是   | `string`                | 寄件区县。               |
| `contactAddress`       | 是   | `string`                | 寄件详细地址。           |
| `contactAddressDetail` | 是   | `string`                | 寄件补充地址。           |
| `startStation`         | 是   | `string`                | 始发网点编码。           |
| `startStationName`     | 是   | `string`                | 始发网点名称。           |
| `acceptDept`           | 是   | `string`                | 受理部门编码。           |
| `shipperNumber`        | 是   | `string`                | 寄件客户编号。           |
| `pickupManId`          | 是   | `string`                | 取件员 ID。              |
| `dispatchFlag`         | 是   | 枚举："Y"、"N"          | 是否派送或上门取件标识。 |
| `passwordSigning`      | 是   | 枚举："Y"、"N"          | 是否密码签收。           |
| `receive`              | 是   | `ExpressReceiveOrder[]` | 收件订单明细列表。       |
