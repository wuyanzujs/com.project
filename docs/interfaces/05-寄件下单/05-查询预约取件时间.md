# 查询预约取件时间

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                               |
| --------- | ------------------------------------------------------------------ |
| 用途      | 查询可预约上门时间和揽收网点。                                     |
| 源码方法  | `expressApi.queryPickupTime`                                       |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:36`       |
| HTTP 方法 | `POST`                                                             |
| 接口路径  | `/gwapi/orderService/eco/order/dispatchTime/pilotCityDispatchTime` |
| 登录态    | 默认是                                                             |
| 请求类型  | `ExpressPickupTimeRequest`                                         |
| 响应类型  | `ExpressPickupTimeResponse`                                        |

## 请求字段

类型：`ExpressPickupTimeRequest`。

| 字段                   | 必填 | 类型                 | 说明                                                |
| ---------------------- | ---- | -------------------- | --------------------------------------------------- |
| `sysCode`              | 否   | `string`             | 系统来源编码，用于区分 App、小程序、H5 等调用来源。 |
| `provinceName`         | 是   | `string`             | 省份名称。                                          |
| `provinceCode`         | 否   | `string`             | 省份编码。                                          |
| `cityName`             | 是   | `string`             | 城市名称。                                          |
| `cityCode`             | 否   | `string`             | 城市编码。                                          |
| `countyName`           | 是   | `string`             | 区县名称。                                          |
| `countyCode`           | 否   | `string`             | 区县编码。                                          |
| `townName`             | 否   | `string`             | 乡镇或街道名称。                                    |
| `townCode`             | 否   | `string`             | 乡镇或街道编码。                                    |
| `address`              | 是   | `string`             | 详细地址。                                          |
| `weight`               | 是   | `number`             | 重量。                                              |
| `volume`               | 是   | `number`             | 体积。                                              |
| `goodsNumber`          | 否   | `number`             | 货物件数。                                          |
| `originalsaddress`     | 否   | `string`             | 地址字段。                                          |
| `originalsStreet`      | 否   | `string`             | 始发街道或乡镇。                                    |
| `priceTimeProductCode` | 否   | `ExpressProductCode` | 金额或费用相关字段。                                |
| `source`               | 是   | `0`                  | 来源标识。                                          |

## 响应字段

类型：`ExpressPickupTimeResponse`。

| 字段               | 必填 | 类型                         | 说明                       |
| ------------------ | ---- | ---------------------------- | -------------------------- |
| `deptCode`         | 否   | `string`                     | 部门或网点编码。           |
| `deptName`         | 否   | `string`                     | 部门或网点名称。           |
| `startTime`        | 否   | `string`                     | 开始时间。                 |
| `endTime`          | 否   | `string`                     | 结束时间。                 |
| `serviceTime`      | 是   | `string`                     | 时间或日期相关字段。       |
| `opening`          | 是   | `boolean`                    | 是否在营业或可预约时间内。 |
| `pickPeriodTime`   | 否   | `number`                     | 时间或日期相关字段。       |
| `openingMessage`   | 否   | `string`                     | 可预约时间提示文案。       |
| `openingList`      | 是   | `ExpressPickupTimeOpening[]` | 列表字段。                 |
| `blankOpeningList` | 否   | `ExpressPickupTimeOpening[]` | 列表字段。                 |
| `nightOpeningList` | 否   | `ExpressPickupTimeOpening[]` | 列表字段。                 |
