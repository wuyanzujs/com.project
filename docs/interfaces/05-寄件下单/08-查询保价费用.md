# 查询保价费用

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                               |
| --------- | ------------------------------------------------------------------ |
| 用途      | 查询保价或定额保障费用。                                           |
| 源码方法  | `expressApi.queryInsurancePrice`                                   |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:69`       |
| HTTP 方法 | `POST`                                                             |
| 接口路径  | `/gwapi/pricetimeService/eco/fixedProtection/queryFixedProtection` |
| 登录态    | 默认是                                                             |
| 请求类型  | `ExpressInsurancePriceRequest`                                     |
| 响应类型  | `ExpressInsurancePriceResponse`                                    |

## 请求字段

类型：`ExpressInsurancePriceRequest`。

| 字段               | 必填 | 类型                                                                                                           | 说明                                                           |
| ------------------ | ---- | -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| `pricingEntryCode` | 是   | `"BF"`                                                                                                         | 计价项编码。                                                   |
| `productCode`      | 是   | 枚举：空字符串、"PACKAGE"、"DEAP"、"RCP"、"PCP"、"DCZP"、"DJBK"、"DJTK"、"XJBK"、"XJTK"、"XJTH"、"DJTH"、"YTY" | 产品编码。                                                     |
| `statements`       | 是   | string[] 或 number[]                                                                                           | 报价计算所需的声明值列表，数组元素是后端计价规则需要的声明值。 |
| `subType`          | 是   | `string`                                                                                                       | 子类型。                                                       |
| `weight`           | 是   | `number`                                                                                                       | 重量。                                                         |
| `volume`           | 是   | `number`                                                                                                       | 体积。                                                         |

## 响应字段

类型：`ExpressInsurancePriceResponse`。

| 字段                  | 必填 | 类型                     | 说明                 |
| --------------------- | ---- | ------------------------ | -------------------- |
| `data`                | 否   | `Record<string, number>` | 业务数据。           |
| `price`               | 否   | `number`                 | 金额或费用相关字段。 |
| `fixedProtectionName` | 否   | `string`                 | 定额保障名称。       |
