# 查询价格时效

返回 [[05-寄件下单/00-模块总览|寄件下单接口]]，也可以回到 [[00-接口总览]]。

## 接口说明

| 项目      | 说明                                                           |
| --------- | -------------------------------------------------------------- |
| 用途      | 查询价格时效，返回可用产品、预计到达时间、费用明细和计费重量。 |
| 源码方法  | `expressApi.queryFreight`                                      |
| 源码位置  | `apps/com.deppon.app/src/services/express/express.api.ts:25`   |
| HTTP 方法 | `POST`                                                         |
| 接口路径  | `/gwapi/pricetimeService/eco/pricetime/queryPriceTime`         |
| 登录态    | 默认是                                                         |
| 请求类型  | `ExpressFreightRequest`                                        |
| 响应类型  | `ExpressProductQuote[]`                                        |

## 请求字段

类型：`ExpressFreightRequest`。

| 字段                | 必填 | 类型                                                                                                           | 说明                 |
| ------------------- | ---- | -------------------------------------------------------------------------------------------------------------- | -------------------- |
| `channel`           | 否   | `string`                                                                                                       | 渠道编码。           |
| `insuredAmount`     | 是   | `number`                                                                                                       | 保价金额。           |
| `originalsStreet`   | 是   | `string`                                                                                                       | 始发街道或乡镇。     |
| `receiverAddress`   | 是   | `string`                                                                                                       | 收件详细地址。       |
| `originalsaddress`  | 是   | `string`                                                                                                       | 地址字段。           |
| `shipperAddress`    | 是   | `string`                                                                                                       | 地址字段。           |
| `reciveLoanType`    | 是   | 枚举：空字符串、"R1"、"R3"                                                                                     | 代收货款类型。       |
| `returnBillType`    | 是   | 枚举："NONE"、"FAX"、"ORIGINAL"、"ONLINE"                                                                      | 回单类型。           |
| `receiveMethod`     | 是   | 枚举：空字符串、"DELIVER_NOUP"、"SELF_PICKUP"、"DELIVER_UP"、"LARGE_DELIVER_UP"                                | 收货或派送方式。     |
| `reviceMoneyAmount` | 是   | `number`                                                                                                       | 代收货款金额。       |
| `totalVolume`       | 是   | `number`                                                                                                       | 总体积。             |
| `totalWeight`       | 是   | `number`                                                                                                       | 总重量。             |
| `goodsName`         | 否   | `string`                                                                                                       | 货物名称。           |
| `client`            | 是   | `boolean`                                                                                                      | 是否客户端调用标识。 |
| `detail`            | 是   | `boolean`                                                                                                      | 费用明细。           |
| `sendDateTime`      | 是   | `string`                                                                                                       | 寄件或预约时间。     |
| `promotionsCode`    | 否   | `string`                                                                                                       | 编码字段。           |
| `pickUpToDoor`      | 是   | `boolean`                                                                                                      | 是否上门取件。       |
| `passwordSigning`   | 是   | 枚举："Y"、"N"                                                                                                 | 是否密码签收。       |
| `passProductCode`   | 是   | 枚举：空字符串、"PACKAGE"、"DEAP"、"RCP"、"PCP"、"DCZP"、"DJBK"、"DJTK"、"XJBK"、"XJTK"、"XJTH"、"DJTH"、"YTY" | 指定传入的产品编码。 |

## 响应字段

类型：`ExpressProductQuote[]`，数组元素字段如下。

| 字段             | 必填 | 类型                                                                                                           | 说明                 |
| ---------------- | ---- | -------------------------------------------------------------------------------------------------------------- | -------------------- |
| `productName`    | 是   | `string`                                                                                                       | 产品名称。           |
| `producteCode`   | 否   | `string`                                                                                                       | 产品编码。           |
| `omsProductCode` | 是   | 枚举：空字符串、"PACKAGE"、"DEAP"、"RCP"、"PCP"、"DCZP"、"DJBK"、"DJTK"、"XJBK"、"XJTK"、"XJTH"、"DJTH"、"YTY" | OMS 产品编码。       |
| `days`           | 是   | string 或 null                                                                                                 | 预计运输天数。       |
| `daysFormat`     | 否   | string 或 null                                                                                                 | 格式化后的预计时效。 |
| `arriveDate`     | 是   | string 或 null                                                                                                 | 预计到达日期。       |
| `message`        | 是   | string 或 null                                                                                                 | 提示信息。           |
| `label`          | 是   | `string`                                                                                                       | 展示标签。           |
| `totalfee`       | 是   | number 或 null                                                                                                 | 总费用。             |
| `detail`         | 是   | ExpressPriceDetail[] 或 null                                                                                   | 费用明细。           |
| `billWeight`     | 是   | number 或 null                                                                                                 | 计费重量。           |
